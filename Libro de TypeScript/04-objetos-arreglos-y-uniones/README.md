# Capítulo 04 — Objetos, Arreglos y Uniones

En programas reales rara vez trabajas con un solo tipo simple. Lo habitual es
mezclar arreglos, objetos y valores que pueden tomar varias formas validas. Para
eso TypeScript ofrece unions y herramientas de refinamiento.

## Que aprenderas

- arreglos tipados;
- aliases de tipos;
- unions;
- narrowing con `typeof` y `in`;
- por que modelar mejor los datos reduce bugs.
