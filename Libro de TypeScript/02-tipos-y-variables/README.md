# Capítulo 02 — Tipos y Variables

En TypeScript, los tipos ayudan a describir que clase de dato esperas guardar y
usar. Eso permite que el editor y el compilador te avisen cuando una operación
no coincide con lo que el programa necesita.

## Que aprenderas

- tipos primitivos;
- inferencia de tipos;
- anotaciones manuales;
- diferencias entre `any`, `unknown` y tipos concretos;
- por que conviene ser explicito cuando el código lo necesita.

## Que pasa dentro del software

Los tipos de TypeScript no ocupan memoria extra en la ejecución final. Son una
descripción para el compilador y para la persona que mantiene el código.

Eso mejora claridad, autocompletado y detección temprana de errores.
