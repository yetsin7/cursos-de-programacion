# Chapter 27: Open Source Communication

## Descripción

Este capítulo cubre toda la comunicación necesaria para participar en proyectos de código abierto: desde contribuir código hasta reportar bugs y mantener proyectos. Aprenderas las frases, formatos y tono profesional que se usan en la comunidad open source.

## Contenido

| Leccion | Tema | Descripción |
|---------|------|-------------|
| 01 | Contributing to Projects | Como contribuir a proyectos open source en ingles |
| 02 | Issue Reporting | Como reportar bugs y solicitar features |
| 03 | Maintaining Projects | Como mantener y gestionar un proyecto open source |
| 04 | Exercises | Ejercicios prácticos de comunicación open source |

## Objetivos de aprendizaje

- Comunicarte profesionalmente en proyectos open source
- Escribir reportes de bugs claros y útiles
- Contribuir código siguiendo las convenciones del proyecto
- Gestionar issues y pull requests como maintainer
