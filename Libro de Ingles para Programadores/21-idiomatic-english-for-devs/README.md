# Chapter 21: Idiomatic English for Developers

## Descripcion

Este capitulo cubre el ingles idiomatico que todo desarrollador necesita conocer: expresiones, modismos tecnologicos, phrasal verbs comunes en el mundo tech y jerga informal que encontraras en equipos de desarrollo, repositorios open source y comunidades online.

| 4 | Ejercicios | `04_exercises.md` |

## Objetivos de aprendizaje

- Entender y usar modismos comunes en el mundo del desarrollo de software
- Dominar los phrasal verbs mas frecuentes en contextos tecnologicos
- Reconocer y usar jerga informal de manera apropiada
- Comunicarte como un desarrollador nativo en conversaciones tecnicas
