# Prepositions with Tech Terms

## Preposiciones que acompañan términos técnicos

En inglés, muchos verbos y expresiones van acompañados de preposiciones específicas. No puedes elegir libremente la preposición: cada combinación es fija. Estas se llaman **collocations** (colocaciones) y son esenciales en el inglés técnico.

---

## 1. Verbos + preposiciones fijas

### Depend on — Depender de

```text
The frontend depends on the API.
This module depends on three external libraries.
The performance depends on the database configuration.
```

### Consist of — Consistir en / estar compuesto de

```text
The system consists of three microservices.
The team consists of five developers and two QA engineers.
The payload consists of a header and a body.
```

### Result in — Resultar en / causar

```text
The bug results in a crash on iOS.
Ignoring the warning will result in data loss.
The migration resulted in 15 minutes of downtime.
```

### Lead to — Llevar a / conducir a

```text
This approach leads to better performance.
Memory leaks lead to server crashes.
Poor error handling can lead to security vulnerabilities.
```

### Refer to — Referirse a / hacer referencia a

```text
This variable refers to the user object.
The documentation refers to the old API version.
"It" in this context refers to the database connection.
```

## 2. Acciones técnicas comunes

### Listen on (port) — Escuchar en (puerto)

```text
The server listens on port 3000.
The WebSocket server listens on port 8080.
Make sure nothing else is listening on that port.
```

### Connect to — Conectar a

```text
The app connects to the database on startup.
I can't connect to the remote server.
The client connects to the WebSocket endpoint.
```

### Log in to / Log into — Iniciar sesión en

```text
Log in to the admin panel.
I can't log in to the staging server.
Users log into the app with their email.
```

> **Nota:** "Log in" es el verbo (dos palabras). "Login" es el sustantivo/adjetivo.
> "Log in to the system" vs "The login page is broken."

### Sign up for — Registrarse en

```text
Sign up for a free account.
Users can sign up for notifications.
She signed up for the beta program.
```

### Opt in / Opt out — Aceptar / rechazar (participar)

```text
Users must opt in to receive emails.
You can opt out of data collection.
The feature is opt-in by default.
```

## 3. Descripción de tecnologías

### Based on — Basado en

```text
The framework is based on React.
The architecture is based on microservices.
This decision was based on performance benchmarks.
```

### Built with — Construido con

```text
The app is built with Flutter.
The API is built with Node.js and Express.
The dashboard was built with Vue and Tailwind.
```

### Written in — Escrito en

```text
The backend is written in Go.
The tests are written in Python.
The CLI tool is written in Rust.
```

### Powered by — Impulsado por / funciona con

```text
The search is powered by Elasticsearch.
The site is powered by Next.js.
Authentication is powered by Firebase.
```

### Compatible with — Compatible con

```text
The library is compatible with Node 18+.
The plugin is not compatible with the latest version.
This approach is backward-compatible with v2.
```

## 4. Acciones de desarrollo

| Acción | Preposición | Ejemplo |
|--------|-------------|---------|
| work on | trabajar en | I'm working on the login feature. |
| focus on | enfocarse en | Let's focus on performance. |
| switch to | cambiar a | Switch to the develop branch. |
| migrate to/from | migrar a/desde | We migrated from MySQL to PostgreSQL. |
| upgrade to | actualizar a | Upgrade to version 5.0. |
| downgrade to | degradar a | Downgrade to the previous version. |
| contribute to | contribuir a | She contributes to open-source projects. |
| subscribe to | suscribirse a | Subscribe to the event stream. |
| respond to | responder a | The server responds to GET requests. |
| belong to | pertenecer a | This function belongs to the auth module. |
| apply to | aplicar a | This rule applies to all endpoints. |
| account for | representar / tener en cuenta | Cache hits account for 80% of requests. |
| look into | investigar | I'll look into the issue. |
| run into | encontrarse con (problema) | I ran into a dependency conflict. |
| come up with | idear / proponer | She came up with a clever solution. |

## 5. Preposiciones en mensajes de error y documentación

### Comunes en error messages

```text
Failed to connect to the database.
Permission denied for user 'admin'.
Invalid value for parameter 'limit'.
Error at line 42 in file app.js.
Cannot read property of undefined.
Timeout waiting for response from server.
```

### Comunes en documentación

```text
For more information, refer to the docs.
This feature was introduced in version 3.0.
The function takes an array as input.
Pass the token in the Authorization header.
The data is returned in JSON format.
Set the value to true in the config file.
```

## 6. Phrasal verbs técnicos comunes

Los phrasal verbs son verbos + preposición/adverbio que crean un significado nuevo.

| Phrasal verb | Significado | Ejemplo |
|-------------|-------------|---------|
| set up | configurar | Set up the development environment. |
| break down | descomponer / fallar | Break down the task into subtasks. |
| roll out | desplegar gradualmente | We're rolling out the update. |
| roll back | revertir | Roll back to the previous version. |
| spin up | iniciar (servidor/instancia) | Spin up a new container. |
| shut down | apagar | Shut down the server. |
| back up | hacer respaldo | Back up the database before migrating. |
| check out | obtener (rama) | Check out the feature branch. |
| figure out | resolver/descifrar | I need to figure out why it fails. |
| point out | señalar | She pointed out a bug in the logic. |
| carry out | llevar a cabo | We carried out the migration. |
| come across | encontrar por casualidad | I came across a useful library. |
| end up | terminar (resultado) | We ended up rewriting the module. |
| keep up with | mantenerse al día con | It's hard to keep up with new frameworks. |

## 7. Errores comunes de hispanohablantes

| Incorrecto | Correcto | Explicación |
|------------|----------|-------------|
| depends of | depends on | Siempre "depend on" |
| consists in | consists of | Siempre "consist of" |
| connect with the database | connect to the database | "Connect to" para conexiones |
| listen in port 3000 | listen on port 3000 | Puertos usan "on" |
| based in React | based on React | "Based on" para tecnología base |
| written on Python | written in Python | Lenguajes usan "in" |
| I'm working in the bug | I'm working on the bug | "Work on" para tareas |
| log in the system | log in to the system | Necesita "to" para el destino |
| compatible to | compatible with | Siempre "compatible with" |
