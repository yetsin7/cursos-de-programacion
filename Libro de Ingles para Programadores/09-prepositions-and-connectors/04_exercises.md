## Ejercicio 6: Une las oraciones con el conector indicado

1. The test passed. The build failed. (but)
2. The API changed. We need to update the client. (since)
3. Deploy to staging first. We might break production. (otherwise)
4. Install Node. Run npm install. Start the dev server. (first/then/finally)
5. The code is clean. It's well-tested. (not only... but also)
6. The server crashed. We lost some data. (as a result)

## Ejercicio 9: Traduce al inglés

1. El servidor escucha en el puerto 8080.
2. La app está construida con Flutter y escrita en Dart.
3. El build falló porque una dependencia estaba desactualizada.
4. Haz deploy a staging primero; de lo contrario, podemos romper producción.
5. Aunque el código funciona, necesita refactorización.
6. El API es rápida. Además, está bien documentada.
7. Primero, clona el repo. Luego, instala las dependencias. Finalmente, ejecuta el servidor.
8. La caída fue debido a una mala configuración.
9. A menos que los tests pasen, no haré merge del PR.
10. Investiga el problema antes de hacer cambios.

## Ejercicio 10: Situación — Escribe un reporte de incidente

Usa conectores y preposiciones para escribir un breve reporte en inglés sobre esta situación:

- El servidor se cayó a las 2 AM del lunes
- Fue porque la base de datos se quedó sin espacio
- El equipo investigó y encontró el problema en 30 minutos
- Sin embargo, la restauración tardó 2 horas
- Además, 500 usuarios fueron afectados
- Como resultado, van a agregar monitoreo de espacio en disco

The server went down at 2 AM on Monday due to the database running out of disk space. The team investigated and identified the root cause within 30 minutes. However, the full restoration took approximately two hours. In addition, around 500 users were affected by the outage. As a result, the team is going to implement disk space monitoring to prevent similar incidents.
