# Prepositions of Place and Time

## Preposiciones de lugar y tiempo: in, on, at

Las preposiciones son una de las partes más difíciles del inglés para hispanohablantes porque no siempre corresponden con las preposiciones en español. En esta lección aprenderás las reglas para **in**, **on** y **at**, y las preposiciones de movimiento.

---

## 1. Preposiciones de tiempo: in, on, at

### AT — Momentos específicos (horas, momentos del día)

| Uso | Ejemplo | Traducción |
|-----|---------|------------|
| Horas | at 3 PM | a las 3 PM |
| Mediodía/medianoche | at noon / at midnight | al mediodía / a medianoche |
| Momentos del día | at night | en la noche |
| Momentos específicos | at the moment | en este momento |
| Fines de semana (UK) | at the weekend | el fin de semana |

```text
The deployment is scheduled at 10 PM.
The server crashed at midnight.
We have a standup at 9:30 AM.
At the moment, the API is down.
```

### ON — Días y fechas específicas

| Uso | Ejemplo | Traducción |
|-----|---------|------------|
| Días de la semana | on Monday | el lunes |
| Fechas | on March 15 | el 15 de marzo |
| Días especiales | on my first day | en mi primer día |
| Fines de semana (US) | on the weekend | el fin de semana |

```text
We deploy on Tuesdays.
The release is on March 20th.
I started on Monday.
We never deploy on Fridays.
```

### IN — Períodos largos (meses, años, estaciones, partes del día)

| Uso | Ejemplo | Traducción |
|-----|---------|------------|
| Meses | in January | en enero |
| Años | in 2025 | en 2025 |
| Estaciones | in summer | en verano |
| Partes del día | in the morning | en la mañana |
| Períodos | in two weeks | en dos semanas |
| Siglos/décadas | in the 2020s | en los 2020s |

```text
We started the migration in March.
The feature will be ready in two weeks.
I prefer to code in the morning.
The framework was released in 2023.
```

### Resumen visual

```
AT (punto exacto)     ON (día/fecha)       IN (período)
─────●─────           ─────████─────       ─────████████─────
at 3 PM               on Monday            in January
at midnight           on March 15          in 2025
at the moment         on Friday            in the morning
```

## 2. Preposiciones de lugar: in, on, at

### IN — Dentro de algo (contenedor, espacio cerrado)

| Contexto técnico | Ejemplo |
|-----------------|---------|
| Dentro de una estructura | The value is **in** the array. |
| Dentro de un archivo | The bug is **in** the config file. |
| Dentro de un bloque | The error occurs **in** the try block. |
| Dentro de un entorno | It works **in** production. |
| Escrito en un lenguaje | The app is written **in** Python. |
| En una carpeta | The file is **in** the src directory. |

```text
The data is stored in a database.
There's a typo in line 42.
The function is defined in utils.js.
We found a vulnerability in the authentication module.
```

### ON — Sobre una superficie o plataforma

| Contexto técnico | Ejemplo |
|-----------------|---------|
| En un servidor | The app runs **on** the server. |
| En una plataforma | Deploy **on** AWS / **on** Vercel. |
| En un dispositivo | It works **on** mobile. |
| En un sistema operativo | It runs **on** Linux. |
| En una página/pantalla | The button is **on** the home page. |
| En una rama | I'm working **on** the feature branch. |
| En un puerto | The service listens **on** port 3000. |

```text
The API is hosted on AWS.
The app crashes on iOS but works on Android.
The service runs on port 8080.
I'm working on the main branch.
```

### AT — Punto específico (dirección, posición)

| Contexto técnico | Ejemplo |
|-----------------|---------|
| En un índice | The element is **at** index 0. |
| En una URL | The docs are **at** this URL. |
| En una posición | Look **at** line 42. |
| En una empresa | She works **at** Google. |

```text
The error occurs at line 156.
The first element is at index 0.
You can find the documentation at docs.example.com.
He works at a startup.
```

## 3. Preposiciones de movimiento

| Preposición | Significado | Ejemplo técnico |
|-------------|-------------|-----------------|
| **to** | hacia | Push **to** the remote. / Deploy **to** production. |
| **from** | desde | Pull **from** main. / Import **from** the module. |
| **into** | hacia adentro | Import **into** the module. / Insert **into** the database. |
| **out of** | fuera de | Move the logic **out of** the component. |
| **through** | a través de | The data flows **through** the middleware. |
| **across** | a través de (horizontal) | The change propagates **across** all services. |
| **between** | entre (dos) | The data moves **between** client and server. |
| **among** | entre (varios) | Distribute the load **among** the workers. |

### Ejemplos en contexto

```text
Push the changes to the remote repository.
Pull the latest code from the main branch.
Import the component into the page.
We need to move this function out of the controller.
The request passes through three middleware layers.
The update was rolled out across all regions.
Data is exchanged between the frontend and the API.
```

## 4. Preposiciones de lugar comunes en tech

| Frase | Significado | Ejemplo |
|-------|-------------|---------|
| in the cloud | en la nube | The data is stored in the cloud. |
| on the server | en el servidor | The file is on the server. |
| in the database | en la base de datos | The record is in the database. |
| on the frontend / backend | en el frontend / backend | The validation is on the backend. |
| in the browser | en el navegador | It runs in the browser. |
| at the top / bottom | arriba / abajo | The import is at the top of the file. |
| in the console | en la consola | Check the output in the console. |
| on the screen | en la pantalla | The error shows on the screen. |
| in the queue | en la cola | The job is waiting in the queue. |
| on the network | en la red | There's a bottleneck on the network. |

## 5. Errores comunes de hispanohablantes

| Incorrecto | Correcto | Explicación |
|------------|----------|-------------|
| in Monday | on Monday | Días de la semana usan "on" |
| in 3 PM | at 3 PM | Horas usan "at" |
| on January | in January | Meses usan "in" |
| at the array | in the array | Dentro de un contenedor usa "in" |
| in port 3000 | on port 3000 | Puertos usan "on" |
| The app is written on Python | The app is written in Python | Lenguajes usan "in" |
| Deploy in production | Deploy to production | Movimiento usa "to" |
| I work in Google | I work at Google | Empresas usan "at" |

## Resumen

| Preposición | Tiempo | Lugar (tech) |
|-------------|--------|--------------|
| **at** | horas, momentos | índice, URL, línea, empresa |
| **on** | días, fechas | servidor, plataforma, puerto, rama, dispositivo |
| **in** | meses, años, períodos | array, archivo, base de datos, lenguaje, carpeta |
| **to** | — | push to, deploy to, migrate to |
| **from** | — | pull from, import from |
| **into** | — | insert into, import into |
