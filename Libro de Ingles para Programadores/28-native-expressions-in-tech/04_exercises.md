# Exercises: Native Expressions in Tech

## Ejercicio 1: Vocabulario avanzado

Completa las oraciones con la expresion correcta:

**Opciones:** low-hanging fruit, bandwidth, blast radius, guardrails, edge case, workaround, blocker, greenfield, deep dive, scope creep

1. "I don't have ___ for another project this sprint."
2. "Let's tackle the ___ first — easy wins with high impact."
3. "TypeScript adds ___ that prevent common mistakes."
4. "What's the ___ if the payment service goes down?"
5. "We found a ___ until the official fix is released."
6. "This is a ___ project — no legacy code to worry about."
7. "The API being down is a ___ for the entire team."
8. "What happens when the user enters an empty string? That's an ___."
9. "We need to do a ___ into why response times increased."
10. "Watch out for ___ — the client keeps adding new requirements."
