# Yes/No Questions

## Preguntas de sí o no en inglés técnico

Las preguntas de sí/no son las más básicas en inglés. Se responden con **Yes** o **No** y son fundamentales en reuniones, debugging y conversaciones técnicas diarias.

---

## 1. Preguntas con Do / Does (Presente Simple)

Usamos **Do** con: I, you, we, they.
Usamos **Does** con: he, she, it.

### Estructura

```
Do/Does + sujeto + verbo base + complemento?
```

### Ejemplos en contexto de programación

| Pregunta en inglés | Traducción |
|--------------------|------------|
| Do you have write access? | ¿Tienes acceso de escritura? |
| Do we need to update the dependencies? | ¿Necesitamos actualizar las dependencias? |
| Do they use Git for version control? | ¿Ellos usan Git para control de versiones? |
| Does the API return JSON? | ¿La API devuelve JSON? |
| Does this function handle errors? | ¿Esta función maneja errores? |
| Does the server support HTTPS? | ¿El servidor soporta HTTPS? |
| Does it compile without warnings? | ¿Compila sin advertencias? |

> **Nota:** Con **Does**, el verbo principal va en su forma base (sin -s).
> Incorrecto: *Does the API returns JSON?*
> Correcto: **Does the API return JSON?**

---

## 2. Preguntas con Is / Are (Verbo To Be)

Usamos **Is** con: he, she, it (singular).
Usamos **Are** con: you, we, they (plural).

### Estructura

```
Is/Are + sujeto + complemento?
```

### Ejemplos en contexto de programación

| Pregunta en inglés | Traducción |
|--------------------|------------|
| Is the server running? | ¿Está el servidor corriendo? |
| Is this variable initialized? | ¿Está esta variable inicializada? |
| Is the database connected? | ¿Está la base de datos conectada? |
| Is it deployed to production? | ¿Está desplegado en producción? |
| Are the tests passing? | ¿Están pasando los tests? |
| Are you working on the login feature? | ¿Estás trabajando en la funcionalidad de login? |
| Are we ready to merge? | ¿Estamos listos para hacer merge? |
| Are there any open issues? | ¿Hay algún issue abierto? |

---

## 3. Respuestas cortas (Short Answers)

En inglés, no se responde solo con "Yes" o "No". Se agrega el auxiliar para sonar natural y educado.

### Con Do / Does

| Pregunta | Respuesta afirmativa | Respuesta negativa |
|----------|---------------------|--------------------|
| Do you have access? | Yes, I do. | No, I don't. |
| Does it work? | Yes, it does. | No, it doesn't. |
| Do they use Docker? | Yes, they do. | No, they don't. |
| Does she know Python? | Yes, she does. | No, she doesn't. |

### Con Is / Are

| Pregunta | Respuesta afirmativa | Respuesta negativa |
|----------|---------------------|--------------------|
| Is the server running? | Yes, it is. | No, it isn't. |
| Are the tests passing? | Yes, they are. | No, they aren't. |
| Is it ready? | Yes, it is. | No, it's not. |
| Are you available? | Yes, I am. | No, I'm not. |

> **Importante:** Nunca se contrae la respuesta afirmativa corta.
> Incorrecto: *Yes, it's.* / *Yes, I'm.*
> Correcto: **Yes, it is.** / **Yes, I am.**

---

## 4. Preguntas con Can / Will / Have

Otros auxiliares comunes en preguntas técnicas:

| Pregunta | Traducción |
|----------|------------|
| Can you review my PR? | ¿Puedes revisar mi PR? |
| Can I push to main? | ¿Puedo hacer push a main? |
| Will this break the build? | ¿Esto romperá el build? |
| Will it be ready by Friday? | ¿Estará listo para el viernes? |
| Have you tested this locally? | ¿Has probado esto localmente? |
| Has the deploy finished? | ¿Ha terminado el deploy? |

### Respuestas cortas

| Pregunta | Sí | No |
|----------|----|----|
| Can you help? | Yes, I can. | No, I can't. |
| Will it work? | Yes, it will. | No, it won't. |
| Have you finished? | Yes, I have. | No, I haven't. |

## 5. Preguntas comunes en el día a día del desarrollador

```text
Do you have a minute?              - ¿Tienes un minuto?
Does this look right to you?       - ¿Esto te parece correcto?
Is anyone working on this bug?     - ¿Alguien está trabajando en este bug?
Are we still on track for the release? - ¿Seguimos en camino para el release?
Can you share your screen?         - ¿Puedes compartir tu pantalla?
Have you seen this error before?   - ¿Has visto este error antes?
Does the CI pipeline pass?         - ¿Pasa el pipeline de CI?
Is there a workaround?             - ¿Hay algún workaround?
```

---

## 6. Errores comunes de hispanohablantes

| Incorrecto | Correcto | Explicación |
|------------|----------|-------------|
| The API returns JSON? | Does the API return JSON? | Falta el auxiliar "Does" |
| Does the server is running? | Is the server running? | No se mezclan auxiliares |
| Do you can help me? | Can you help me? | "Can" ya es auxiliar, no necesita "Do" |
| Yes, it's. | Yes, it is. | No se contrae en respuestas cortas afirmativas |
| Are works the tests? | Are the tests working? | El orden es: auxiliar + sujeto + verbo |

---

## Resumen

| Tipo | Auxiliar | Ejemplo |
|------|----------|---------|
| Presente simple (acciones) | Do / Does | Does the API return JSON? |
| Estado / descripción | Is / Are | Is the server running? |
| Habilidad / permiso | Can | Can you review my code? |
| Futuro | Will | Will this fix the bug? |
| Experiencia / completado | Have / Has | Have you deployed it? |
