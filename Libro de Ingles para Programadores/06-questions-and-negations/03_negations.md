# Negations

## Negaciones en inglés técnico

Saber negar correctamente es fundamental para reportar errores, describir problemas y comunicar limitaciones en tu trabajo diario como programador.

---

## 1. Negaciones con Do / Does (Presente Simple)

Usamos **don't** (do not) con: I, you, we, they.
Usamos **doesn't** (does not) con: he, she, it.

### Estructura

```
Sujeto + don't/doesn't + verbo base + complemento
```

### Ejemplos en programación

| Negación | Traducción |
|----------|------------|
| I don't understand the error message. | No entiendo el mensaje de error. |
| I don't have access to the repository. | No tengo acceso al repositorio. |
| We don't use jQuery anymore. | Ya no usamos jQuery. |
| They don't follow the coding standards. | Ellos no siguen los estándares de código. |
| The test doesn't pass. | El test no pasa. |
| This function doesn't return a value. | Esta función no devuelve un valor. |
| The API doesn't support pagination. | La API no soporta paginación. |
| It doesn't work on my machine. | No funciona en mi máquina. |

> **Recuerda:** Con **doesn't**, el verbo va en forma base (sin -s).
> Incorrecto: *It doesn't works.*
> Correcto: **It doesn't work.**

## 2. Negaciones con Is / Are (Verbo To Be)

### Contracciones

| Forma completa | Contracción | Contracción alternativa |
|---------------|-------------|------------------------|
| is not | isn't | it's not / he's not |
| are not | aren't | they're not / we're not |
| am not | — | I'm not |

### Ejemplos en programación

| Negación | Traducción |
|----------|------------|
| The server isn't responding. | El servidor no está respondiendo. |
| This variable isn't defined. | Esta variable no está definida. |
| The tests aren't passing. | Los tests no están pasando. |
| We aren't ready to deploy. | No estamos listos para hacer deploy. |
| I'm not sure about this approach. | No estoy seguro de este enfoque. |
| It isn't compatible with the latest version. | No es compatible con la última versión. |
| The endpoints aren't documented. | Los endpoints no están documentados. |

---

## 3. Negaciones con Can / Can't

**Can't** (cannot) expresa imposibilidad o incapacidad.

| Negación | Traducción |
|----------|------------|
| I can't access the database. | No puedo acceder a la base de datos. |
| I can't reproduce the bug. | No puedo reproducir el bug. |
| We can't deploy on Friday. | No podemos hacer deploy el viernes. |
| The user can't log in. | El usuario no puede iniciar sesión. |
| You can't push directly to main. | No puedes hacer push directo a main. |
| It can't handle more than 1000 requests. | No puede manejar más de 1000 solicitudes. |

> **Forma formal:** "cannot" se escribe como una sola palabra.
> "The system cannot process this request."

---

## 4. Negaciones con Will / Won't

**Won't** (will not) expresa que algo no sucederá en el futuro, o que algo se rehúsa a funcionar.

| Negación | Traducción |
|----------|------------|
| It won't compile. | No compila. (Se rehúsa a compilar) |
| The container won't start. | El contenedor no arranca. |
| This won't work in production. | Esto no va a funcionar en producción. |
| The tests won't pass until we fix the mock. | Los tests no van a pasar hasta que arreglemos el mock. |
| I won't be able to finish by Friday. | No voy a poder terminar para el viernes. |
| The feature won't be ready for the next sprint. | La funcionalidad no estará lista para el próximo sprint. |

> **Nota cultural:** "Won't" a veces implica que algo "se rehúsa" a funcionar, como si tuviera voluntad propia. Es muy común en tech: "The server won't start" (el servidor no quiere arrancar).

---

## 5. Negaciones con Have / Has

**Haven't** (have not) y **hasn't** (has not) se usan con el presente perfecto.

| Negación | Traducción |
|----------|------------|
| I haven't finished the code review. | No he terminado la revisión de código. |
| She hasn't pushed her changes yet. | Ella no ha subido sus cambios todavía. |
| We haven't tested this on mobile. | No hemos probado esto en móvil. |
| The build hasn't completed yet. | El build no ha completado todavía. |
| They haven't responded to the PR. | No han respondido al PR. |

## 6. Contracciones vs. formas formales

En el día a día usamos contracciones. En documentación formal, usamos la forma completa.

| Contracción (informal) | Forma completa (formal) |
|------------------------|------------------------|
| don't | do not |
| doesn't | does not |
| isn't | is not |
| aren't | are not |
| can't | cannot |
| won't | will not |
| haven't | have not |
| hasn't | has not |
| shouldn't | should not |
| wouldn't | would not |
| couldn't | could not |

### Cuándo usar cada forma

```text
Conversación / Slack / Chat:
  "It doesn't work."
  "I can't access the repo."

Documentación técnica / Reportes de bugs:
  "The function does not return the expected value."
  "The system cannot handle concurrent requests."

Correos formales:
  "We will not be able to meet the deadline."
  "This approach is not recommended."
```

## 7. Otras negaciones útiles

### Never / No / None / Nothing / Nobody

| Palabra | Significado | Ejemplo |
|---------|-------------|---------|
| never | nunca | We never deploy on Fridays. |
| no | ningún | There are no errors in the log. |
| none | ninguno | None of the tests failed. |
| nothing | nada | Nothing happens when I click the button. |
| nobody / no one | nadie | Nobody has reviewed this PR yet. |
| nowhere | en ningún lugar | The variable is declared nowhere in the file. |

> **Importante:** En inglés NO se usa doble negación.
> Incorrecto: *I don't have nothing.*
> Correcto: **I don't have anything.** / **I have nothing.**

---

## 8. Frases negativas comunes en el trabajo diario

```text
That's not going to work.          - Eso no va a funcionar.
I don't think this is the right approach. - No creo que este sea el enfoque correcto.
This shouldn't take long.          - Esto no debería tardar mucho.
We shouldn't merge without tests.  - No deberíamos hacer merge sin tests.
I wouldn't recommend that.         - Yo no recomendaría eso.
It's not a priority right now.     - No es una prioridad ahora mismo.
That doesn't make sense.           - Eso no tiene sentido.
I'm not blocked, but I need clarification. - No estoy bloqueado, pero necesito clarificación.
```

## 9. Errores comunes de hispanohablantes

| Incorrecto | Correcto | Explicación |
|------------|----------|-------------|
| I no understand. | I don't understand. | Se necesita el auxiliar "don't" |
| It doesn't works. | It doesn't work. | Después de "doesn't" va verbo base |
| I don't can do it. | I can't do it. | "Can" es auxiliar, se niega directamente |
| I have not nothing. | I have nothing. / I don't have anything. | No usar doble negación |
| The server no is running. | The server isn't running. | "Not" va después del verbo to be |
| He don't know. | He doesn't know. | Tercera persona usa "doesn't" |
