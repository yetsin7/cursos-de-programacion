# Capítulo 06: Questions & Negations

## Preguntas y negaciones en contexto de programación

Este capítulo te enseñará a formular preguntas y construir oraciones negativas en inglés, usando situaciones reales del mundo del desarrollo de software.

1. **Yes/No Questions** — Preguntas con Do/Does/Is/Are y respuestas cortas
2. **WH Questions** — Preguntas con What, Where, When, Why, How, Who, Which
3. **Negations** — Formas negativas: don't, doesn't, isn't, can't, won't

## Objetivo del capítulo

Al terminar este capítulo podrás:

- Hacer preguntas técnicas correctamente en inglés
- Responder con respuestas cortas apropiadas
- Usar palabras interrogativas para obtener información específica
- Construir oraciones negativas para reportar errores y problemas
- Comunicarte efectivamente en reuniones, code reviews y chats técnicos
