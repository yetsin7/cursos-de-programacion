# Chapter 11: Programming Vocabulary

## Vocabulario de Programacion

En este capitulo aprenderemos el vocabulario esencial que todo programador necesita
dominar en ingles. Cubriremos desde los tipos de datos basicos hasta la terminologia
de desarrollo web y movil.

## Contenido

| Leccion | Tema | Descripcion |
|---------|------|-------------|
| 01 | Data Types Vocabulary | Tipos de datos: string, integer, boolean, array, etc. |
| 02 | OOP Vocabulary | Programacion orientada a objetos: class, object, inheritance, etc. |
| 03 | Web and Mobile Vocabulary | Terminologia web y movil: API, frontend, backend, widget, etc. |
| 04 | Exercises | Ejercicios practicos para reforzar el vocabulario |

## Objetivos

Al completar este capitulo seras capaz de:

- Nombrar y pronunciar correctamente los tipos de datos en ingles
- Usar vocabulario de OOP en conversaciones tecnicas
- Entender y usar terminologia de desarrollo web y movil
- Comunicarte con confianza sobre conceptos de programacion en ingles
