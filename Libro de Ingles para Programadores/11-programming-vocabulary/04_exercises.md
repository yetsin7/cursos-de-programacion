## Ejercicio 6: Corrige el Error

Cada oracion tiene un error de vocabulario. Identificalo y corrigelo.

1. "The function returns a void string." →
2. "Use a dictionary to store ordered items without keys." →
3. "The singleton creates a new instance every time." →
4. "The getter assigns a new value to the property." →
5. "Deploy the app to the frontend server." →

<details>
<summary>Respuestas</summary>

1. Error: "void string" - void significa sin retorno. Correccion: "The function returns void." o "The function returns a string."
2. Error: dictionary es para clave-valor. Correccion: "Use a list to store ordered items."
3. Error: singleton solo crea una instancia. Correccion: "The singleton returns the same instance every time."
4. Error: getter obtiene, no asigna. Correccion: "The setter assigns a new value to the property."
5. Error: se despliega al backend/servidor. Correccion: "Deploy the app to the backend server." o "Deploy the app to production."
</details>

## Ejercicio 8: Vocabulario Cruzado

Responde en ingles:

1. What is the difference between a **class** and an **object**?
2. What is the difference between **null** and **undefined**?
3. What does a **middleware** do?
4. When would you use a **set** instead of a **list**?
5. What is the difference between a **getter** and a **setter**?

<details>
<summary>Respuestas Sugeridas</summary>

1. A class is a blueprint/template; an object is a concrete instance of that class.
2. Null means intentionally empty; undefined means a variable was declared but never assigned a value.
3. A middleware processes requests between the client and the server, often for authentication or logging.
4. Use a set when you need a collection with no duplicate values.
5. A getter retrieves a property value; a setter assigns a new value to a property.
</details>
