# Adverbs in Tech Context

## Adverbios en contexto técnico

Los adverbios modifican verbos, adjetivos u otros adverbios. Nos permiten describir **cómo**, **cuándo**, **dónde** y **con qué frecuencia** ocurre algo. Son fundamentales para comunicar detalles técnicos con precisión.

---

## 1. Adverbios de modo (How? - ¿Cómo?)

Describen la manera en que se realiza una acción. Muchos se forman agregando **-ly** al adjetivo.

| Adverbio | Significado | Ejemplo |
|----------|-------------|---------|
| quickly | rápidamente | The server responds quickly. |
| slowly | lentamente | The page loads slowly on mobile. |
| correctly | correctamente | The function works correctly now. |
| efficiently | eficientemente | The algorithm sorts data efficiently. |
| recursively | recursivamente | The function calls itself recursively. |
| asynchronously | asincrónicamente | The data loads asynchronously. |
| automatically | automáticamente | Tests run automatically on push. |
| manually | manualmente | I deployed it manually. |
| properly | correctamente / apropiadamente | The error isn't handled properly. |
| securely | de forma segura | Store passwords securely. |
| seamlessly | sin problemas / fluidamente | The migration went seamlessly. |
| independently | independientemente | Each module works independently. |

### Formación: adjetivo -> adverbio

| Adjetivo | Adverbio | Regla |
|----------|----------|-------|
| quick | quickly | + ly |
| efficient | efficiently | + ly |
| automatic | automatically | + ally |
| easy | easily | y -> ily |
| simple | simply | le -> ly |

> **Excepciones importantes:**
> - **fast** -> **fast** (no *fastly*)
> - **hard** -> **hard** (no *hardly*, que significa "apenas")
> - **good** -> **well** (no *goodly*)

## 2. Adverbios de lugar (Where? - ¿Dónde?)

| Adverbio | Significado | Ejemplo |
|----------|-------------|---------|
| locally | localmente | Run the server locally. |
| remotely | remotamente | The team works remotely. |
| globally | globalmente | The variable is defined globally. |
| internally | internamente | The API is used internally. |
| externally | externamente | The service connects externally. |
| here | aquí | The bug is right here. |
| there | allí | The config file should go there. |
| everywhere | en todas partes | This pattern is used everywhere. |
| nowhere | en ningún lugar | That function is used nowhere. |

```text
We store data locally, not in the cloud.
The package is available globally after installation.
The variable is declared here but used there.
```

## 3. Adverbios de tiempo (When? - ¿Cuándo?)

| Adverbio | Significado | Ejemplo |
|----------|-------------|---------|
| temporarily | temporalmente | The fix is only temporary. We applied it temporarily. |
| permanently | permanentemente | The data was permanently deleted. |
| immediately | inmediatamente | The server restarts immediately. |
| eventually | eventualmente | The cache will eventually expire. |
| already | ya | I've already pushed the fix. |
| recently | recientemente | The API was recently updated. |
| currently | actualmente | We're currently using Node 20. |
| previously | previamente | This was previously handled by a cron job. |
| soon | pronto | The feature will be available soon. |

---

## 4. Adverbios de frecuencia (How often? - ¿Con qué frecuencia?)

| Adverbio | Frecuencia | Ejemplo |
|----------|-----------|---------|
| always | siempre (100%) | Always write unit tests. |
| usually | usualmente (80%) | We usually deploy on Tuesdays. |
| often | a menudo (60%) | This error occurs often. |
| sometimes | a veces (40%) | Sometimes the API times out. |
| rarely | raramente (10%) | The server rarely goes down. |
| never | nunca (0%) | Never commit secrets to the repo. |

---

## 5. Adverbios de grado (How much? - ¿Cuánto?)

Modifican adjetivos u otros adverbios para indicar intensidad.

| Adverbio | Significado | Ejemplo |
|----------|-------------|---------|
| very | muy | The response is very slow. |
| really | realmente | This library is really useful. |
| quite | bastante | The code is quite readable. |
| fairly | bastante / razonablemente | The setup is fairly simple. |
| extremely | extremadamente | The query is extremely complex. |
| completely | completamente | The feature is completely broken. |
| slightly | ligeramente | The latency increased slightly. |
| barely | apenas | The server can barely handle the load. |
| too | demasiado | The function is too long. |
| enough | suficiente | The test coverage isn't good enough. |

---

## 6. Posición de los adverbios en la oración

La posición del adverbio depende de su tipo:

### Adverbios de modo: después del verbo o al final

```text
The function executes quickly.
The server handles requests efficiently.
```

### Adverbios de frecuencia: antes del verbo principal (después de "be")

```text
We always run tests before merging.
The server is usually stable.
I never push directly to main.
```

### Adverbios de grado: antes del adjetivo/adverbio que modifican

```text
The code is very clean.
It runs extremely fast.
The API is too slow.
```

### Adverbios de tiempo: al inicio o al final de la oración

```text
Recently, we migrated to TypeScript.
The feature was deployed yesterday.
I already fixed that bug.
```

---

## 7. Adverbios que no terminan en -ly

Algunos adverbios importantes tienen la misma forma que el adjetivo:

| Palabra | Como adjetivo | Como adverbio |
|---------|--------------|---------------|
| fast | a fast server | The server runs fast. |
| hard | a hard problem | I worked hard on this. |
| early | an early release | We released early. |
| late | a late commit | He committed late. |
| straight | a straight line | Go straight to production. |

> **Cuidado con "hardly":**
> - **hard** (adverbio) = con esfuerzo: "I worked hard."
> - **hardly** (adverbio) = apenas: "I can hardly see the text."

## 8. Frases comunes con adverbios en tech

```text
The code compiles successfully.       - El código compila exitosamente.
The test consistently fails.          - El test falla consistentemente.
The API responds intermittently.      - La API responde intermitentemente.
The data is stored encrypted locally. - Los datos se almacenan encriptados localmente.
The app gracefully handles errors.    - La app maneja errores de forma elegante.
The system automatically scales.      - El sistema escala automáticamente.
```

## Resumen

| Tipo | Ejemplos | Posición típica |
|------|----------|-----------------|
| Modo | quickly, correctly, efficiently | Después del verbo / al final |
| Lugar | locally, remotely, globally | Al final |
| Tiempo | recently, already, soon | Inicio o final |
| Frecuencia | always, usually, never | Antes del verbo principal |
| Grado | very, too, extremely | Antes del adjetivo/adverbio |
