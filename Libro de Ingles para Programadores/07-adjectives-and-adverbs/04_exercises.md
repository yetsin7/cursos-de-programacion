## Ejercicio 7: Describe la situación

Escribe una oración en inglés para cada situación usando adjetivos y adverbios apropiados.

1. El código de un compañero es muy difícil de leer y no tiene comentarios.
2. La API responde en menos de 100ms incluso con mucho tráfico.
3. Los tests fallan a veces sí y a veces no (son inestables).

6. La migración de la base de datos fue sin problemas.

## Ejercicio 8: Code review - Elige la frase apropiada

Para cada situación de code review, elige la mejor frase.

**1. La función tiene 500 líneas:**
- a) The function is slightly long.
- b) The function is way too long — consider splitting it.
- c) The function is perfectly fine.

**2. Los nombres de variables son "a", "b", "x":**
- a) The naming is very creative.
- b) The naming could be more descriptive.
- c) The naming is well-documented.

**3. El código tiene tests completos:**
- a) The test coverage is comprehensive.
- b) The test coverage is insufficient.
- c) The test coverage is poorly written.

**4. El manejo de errores solo hace `catch(e) {}`:**
- a) The error handling is robust.
- b) The error handling is graceful.
- c) The errors are silently swallowed — add proper handling.
