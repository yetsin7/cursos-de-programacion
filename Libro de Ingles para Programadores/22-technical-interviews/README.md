# Chapter 22: Technical Interviews

## Descripción

Este capítulo te prepara para entrevistas técnicas en ingles. Desde la presentación personal hasta discusiones de system design, aprenderas las frases, estructuras y patrones que necesitas para destacar en entrevistas con empresas internacionales.

| 4 | Lenguaje de system design | `04_system_design_language.md` |
| 5 | Ejercicios | `05_exercises.md` |

## Objetivos de aprendizaje

- Estructurar una presentación personal profesional en ingles
- Responder preguntas de comportamiento con el método STAR
- Comunicar tu razonamiento técnico con claridad
- Usar vocabulario apropiado en entrevistas de system design
