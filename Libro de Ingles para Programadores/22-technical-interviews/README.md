# Chapter 22: Technical Interviews

## Descripcion

Este capitulo te prepara para entrevistas tecnicas en ingles. Desde la presentacion personal hasta discusiones de system design, aprenderas las frases, estructuras y patrones que necesitas para destacar en entrevistas con empresas internacionales.

## Objetivos de aprendizaje

- Estructurar una presentacion personal profesional en ingles
- Responder preguntas de comportamiento con el metodo STAR
- Comunicar tu razonamiento tecnico con claridad
- Usar vocabulario apropiado en entrevistas de system design
