# Ejercicios Integrales - Capítulo 04: Verbos Comunes y Tiempo Presente

## Introducción

Estos ejercicios cubren los verbos esenciales de programación, presente simple y presente continuo. Intenta completarlos sin revisar las lecciones.

---

## Sección D: Simple vs. Continuo

### Ejercicio D1: Elige la forma correcta

1. The server (runs / is running) on port 3000. [verdad general]
2. I (debug / am debugging) the payment module right now.
3. She always (reviews / is reviewing) PRs before lunch.
4. We (deploy / are deploying) to staging at this moment.
5. The function (returns / is returning) a promise. [siempre]
6. The tests (fail / are failing) right now but they usually (pass / are passing).
7. He (works / is working) from home this week.
8. I (use / am using) Python for data analysis. [en general]

### Ejercicio D2: Completa con simple o continuo

1. Right now, the CI pipeline ___ (build) the project.
2. The API ___ (support) GraphQL and REST. [general]
3. I ___ (refactor) the auth module this sprint.
4. She normally ___ (start) coding at 9 AM.
5. At the moment, the database ___ (process) a large migration.
6. We always ___ (write) integration tests for new endpoints.

## Sección F: Situaciones de standup

Escribe lo que dirías en un standup en cada situación:
