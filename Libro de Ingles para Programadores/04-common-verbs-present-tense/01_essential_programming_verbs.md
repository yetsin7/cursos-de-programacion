# Verbos Esenciales de Programación

## Introducción

Estos son los verbos que más usarás como programador en inglés. Cada uno incluye su pronunciación aproximada, significado en español y una oración de ejemplo. Memorizar estos verbos es una de las inversiones más rentables para tu carrera.

---

## Operaciones CRUD y de datos

| Verbo | Pronunciación | Español | Ejemplo |
|-------|--------------|---------|---------|
| **create** | /kri-éit/ | crear | We **create** a new instance of the class |
| **read** | /ríid/ | leer | The function **reads** data from the file |
| **update** | /ap-déit/ | actualizar | The API **updates** the user profile |
| **delete** | /di-lít/ | eliminar | The script **deletes** expired sessions |
| **write** | /ráit/ | escribir | The logger **writes** to a file |
| **save** | /séiv/ | guardar | The app **saves** data to local storage |
| **store** | /stór/ | almacenar | The database **stores** user records |
| **fetch** | /fétch/ | obtener/traer | The component **fetches** data from the API |
| **get** | /guét/ | obtener | This method **gets** the current user |
| **set** | /sét/ | establecer | The function **sets** the default value |
| **insert** | /in-sért/ | insertar | The query **inserts** a new row |
| **remove** | /ri-múuv/ | remover | The filter **removes** duplicates |

## Consejo de estudio

Practica estos verbos en grupos temáticos. No intentes memorizar todos de una vez. Enfócate en un grupo por día:

1. **Día 1:** CRUD (create, read, update, delete, write, save)
2. **Día 2:** Ejecución (run, compile, build, install, start, stop)
3. **Día 3:** Git (push, pull, merge, commit, clone, branch)
4. **Día 4:** Debug (debug, fix, throw, catch, handle, log, test)
5. **Día 5:** Transformación (parse, format, iterate, sort, filter, map)
6. **Día 6:** Arquitectura (import, export, return, deploy, initialize, render)

> **Tip:** Cada vez que uses uno de estos verbos en tu trabajo diario, di la oración completa en inglés en voz alta.
