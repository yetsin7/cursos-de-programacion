# Presente Continuo (Present Continuous)

## Introducción

El presente continuo describe acciones que están **ocurriendo ahora mismo** o **temporalmente en progreso**. En programación es fundamental para describir procesos activos, estados temporales y lo que estás haciendo en este momento.

---

## Estructura

```
Sujeto + am/is/are + Verbo-ing + Complemento
```

| Sujeto | Auxiliar | Verbo-ing | Ejemplo |
|--------|----------|-----------|---------|
| I | **am** | debugging | I **am debugging** the login module |
| You | **are** | working | You **are working** on the API |
| He/She/It | **is** | running | The server **is running** |
| We | **are** | deploying | We **are deploying** to staging |
| They | **are** | testing | They **are testing** the new feature |

---

## Reglas para formar el gerundio (-ing)

| Regla | Ejemplo | Resultado |
|-------|---------|-----------|
| General: agregar -ing | work → work**ing** | debugging, testing, coding |
| Termina en -e muda: quitar -e, agregar -ing | write → writ**ing** | compiling, deploying, creating |
| Termina en consonante-vocal-consonante (monosílabo): duplicar última consonante | run → runn**ing** | stopping, getting, setting |
| Termina en -ie: cambiar -ie por -y, agregar -ing | die → dy**ing** | tying |
| Termina en -ee: solo agregar -ing | see → see**ing** | agreeing |

### Gerundios comunes en programación

| Verbo base | Gerundio | Ejemplo |
|-----------|----------|---------|
| debug | debugging | I am **debugging** a memory leak |
| run | running | The tests are **running** |
| write | writing | She is **writing** documentation |
| compile | compiling | The project is **compiling** |
| deploy | deploying | We are **deploying** to production |
| create | creating | He is **creating** a new branch |
| fix | fixing | I am **fixing** the CSS layout |
| build | building | The CI is **building** the project |
| install | installing | The system is **installing** updates |
| merge | merging | They are **merging** the feature branch |
| render | rendering | The component is **rendering** |
| fetch | fetching | The app is **fetching** data |
| load | loading | The page is **loading** |
| process | processing | The server is **processing** the request |
| download | downloading | The package is **downloading** |

## Forma negativa

```
Sujeto + am/is/are + not + Verbo-ing
```

| Forma completa | Contracción | Ejemplo |
|----------------|-------------|---------|
| I am not | I'm not | I'm not working on that feature |
| It is not | It isn't / It's not | The server isn't responding |
| They are not | They aren't | The tests aren't passing |
| We are not | We aren't | We aren't deploying today |

### Ejemplos en programación

| Inglés | Español |
|--------|---------|
| The server **isn't responding** | El servidor no está respondiendo |
| The tests **aren't passing** | Los tests no están pasando |
| I'**m not working** on that ticket | No estoy trabajando en ese ticket |
| The build **isn't finishing** | El build no está terminando |
| The API **isn't returning** data | La API no está retornando datos |

---

## Forma interrogativa

```
Am/Is/Are + Sujeto + Verbo-ing + Complemento?
```

| Pregunta | Traducción |
|----------|------------|
| **Is** the server **running**? | ¿Está corriendo el servidor? |
| **Are** the tests **passing**? | ¿Están pasando los tests? |
| **Are** you **working** on the bug fix? | ¿Estás trabajando en el bug fix? |
| **Is** the build **compiling**? | ¿Está compilando el build? |
| **Is** anyone **using** the staging server? | ¿Alguien está usando el servidor de staging? |
| What **are** you **working** on? | ¿En qué estás trabajando? |

## Presente Simple vs. Presente Continuo

Esta es una de las diferencias más importantes del inglés. En español a menudo usamos la misma forma, pero en inglés el significado cambia.

| Presente Simple | Presente Continuo |
|----------------|-------------------|
| Hábito, rutina, verdad general | Acción en progreso **ahora** |
| The server **runs** on port 3000 | The server **is running** right now |
| I **write** tests every day | I **am writing** a test right now |
| She **deploys** on Fridays | She **is deploying** at this moment |
| The API **returns** JSON | The API **is returning** an error (ahora) |

### Comparación lado a lado

| Simple (general/siempre) | Continuo (ahora/temporal) |
|--------------------------|---------------------------|
| I **debug** code every day | I **am debugging** the auth module |
| The function **handles** errors | The function **is handling** a timeout |
| We **use** React | We **are using** a new library for this project |
| He **reviews** PRs in the morning | He **is reviewing** my PR right now |
| The tests **pass** in CI | The tests **are passing** locally but not in CI |

### Señales de tiempo (time markers)

| Presente Simple | Presente Continuo |
|----------------|-------------------|
| every day/week/month | right now |
| always, usually, often | at the moment |
| on Mondays, on Fridays | currently |
| in general | today (temporal) |
| normally | this week/sprint |

---

## Uso en standups y reuniones

El presente continuo es el tiempo verbal de los **standups** (reuniones diarias):

| Situación | Ejemplo |
|-----------|---------|
| Lo que estás haciendo | I **am working** on the search feature |
| Bloqueos | The deploy pipeline **is failing** |
| Estado del equipo | We **are waiting** for the design review |
| Progreso | The migration **is running** smoothly |
| Problemas | The database **is taking** too long to respond |
| Planes inmediatos | I **am finishing** the unit tests today |

---

## Errores comunes de hispanohablantes

| Error | Correcto | Explicación |
|-------|----------|-------------|
| ~~I working on it~~ | I **am** working on it | Falta el auxiliar "am" |
| ~~The server is run~~ | The server is **running** | Falta el -ing |
| ~~She is write code~~ | She is **writing** code | Falta el -ing |
| ~~Is running the server?~~ | **Is the server** running? | Orden: Is + sujeto + verbo-ing |
| ~~I am work now~~ | I am **working** now | El verbo necesita -ing |

### Ejercicio 1: Forma el gerundio (-ing)

1. compile → ___
2. run → ___
3. write → ___
4. set → ___
5. create → ___
6. stop → ___
7. deploy → ___
8. get → ___

### Ejercicio 2: Completa con presente continuo

1. I ___ (debug) the authentication module.
2. The server ___ (process) the requests.
3. We ___ (deploy) to production right now.
4. She ___ (review) the pull request.
5. The tests ___ (not / pass) at the moment.
6. ___ you ___ (work) on the API? (pregunta)

### Ejercicio 3: Elige presente simple o continuo

1. The server (run / is running) on port 3000. (verdad general)
2. I (fix / am fixing) a bug right now.
3. She (writes / is writing) clean code. (siempre)
4. We (deploy / are deploying) at this moment.
5. The function (returns / is returning) a promise. (siempre)
6. They (test / are testing) the new feature this week.

### Ejercicio 4: Traduce al inglés

3. Los tests no están pasando.
4. ¿Está compilando el proyecto?

6. Ella está escribiendo documentación ahora. (continuo)
