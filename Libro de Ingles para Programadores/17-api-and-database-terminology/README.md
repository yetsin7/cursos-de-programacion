# Capítulo 17: Terminología de APIs y Bases de Datos

## Descripción

Este capítulo cubre el vocabulario esencial en inglés para trabajar con APIs
y bases de datos. Dominar estos términos es fundamental para leer documentación,
comunicarte con equipos y entender arquitecturas backend.

## Contenido

| Lección | Tema | Descripción |
|---------|------|-------------|
| 01 | API Vocabulary | Términos esenciales de APIs REST, GraphQL y más |
| 02 | Database Vocabulary | Vocabulario de bases de datos SQL y NoSQL |
| 03 | Describing Data Flow | Cómo describir flujos de datos en inglés |
| 04 | Exercises | Ejercicios prácticos del capítulo |

## Objetivos de aprendizaje

Al completar este capítulo serás capaz de:

- Usar correctamente la terminología de APIs en inglés
- Describir operaciones de bases de datos con vocabulario preciso
- Explicar flujos de datos y arquitecturas en inglés
- Leer documentación de APIs y bases de datos sin dificultad

## Prerequisitos

- Capítulos 1-16 completados
- Conocimiento básico de APIs y bases de datos
