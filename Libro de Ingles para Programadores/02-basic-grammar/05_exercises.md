# Ejercicios Integrales - Capítulo 02: Gramática Básica

## Introducción

Estos ejercicios cubren todos los temas del capítulo: estructura SVO, verbo "to be", verbos "to have" y "to do", y "there is/there are". Intenta resolverlos sin consultar las lecciones anteriores.

---

## Sección A: Estructura de oraciones (SVO)

### Ejercicio A1: Ordena las palabras

1. `handles / the / errors / middleware`
2. `tests / writes / she / unit`
3. `every / we / deploy / Friday`
4. `parses / the / JSON / function / the / data`
5. `new / a / created / he / branch`

### Ejercicio A2: Corrige el orden del adjetivo

1. The variable local stores the value. → ___
2. We need a function recursive. → ___
3. She found an error critical. → ___
4. The file temporary was deleted. → ___
5. They use a database relational. → ___

## Sección F: Situaciones reales

### Ejercicio F1: ¿Qué dirías en inglés en estas situaciones?

1. **En un standup:** Quieres decir que estás trabajando en el módulo de autenticación.
2. **En Slack:** Quieres preguntar si el servidor está funcionando.
3. **En un code review:** Quieres señalar que hay un error en la línea 45.
4. **En una reunión:** Quieres decir que la base de datos no tiene índices.
5. **En un PR:** Quieres decir que la función no maneja errores.
6. **En un ticket:** Quieres preguntar si hay tests para esta funcionalidad.

### Ejercicio F2: Corrige los errores

1. "Have a bug in the production server."
2. "The function doesn't returns nothing."
3. "Is a critical error in the deploy."
4. "She have experience with Docker."
5. "The tests doesn't pass on CI."
6. "There is many files to review."
7. "I no have the permissions."
8. "Does the API supports authentication?"
