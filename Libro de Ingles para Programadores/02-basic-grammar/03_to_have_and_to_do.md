# Verbos "To Have" y "To Do" (Tener / Hacer)

## Introducción

Los verbos **"to have"** (tener) y **"to do"** (hacer) tienen un rol doble en inglés: funcionan como **verbos principales** con significado propio, y también como **verbos auxiliares** para formar preguntas y negaciones. Entender esta diferencia es clave.

---

## To Have (Tener)

### Conjugación en presente

| Sujeto | Verbo | Ejemplo |
|--------|-------|---------|
| I | have | I have two monitors |
| You | have | You have access to the repo |
| He/She/It | **has** | The array **has** three elements |
| We | have | We have a deadline tomorrow |
| They | have | They have the credentials |

> **Importante:** La tercera persona singular usa **"has"**, no "haves".

### Como verbo principal en programación

| Inglés | Español |
|--------|---------|
| The array has three elements | El arreglo tiene tres elementos |
| The class has two methods | La clase tiene dos métodos |
| The object has a property called "name" | El objeto tiene una propiedad llamada "name" |
| The function has three parameters | La función tiene tres parámetros |
| The project has no dependencies | El proyecto no tiene dependencias |
| The string has 20 characters | El string tiene 20 caracteres |
| The response has a status code | La respuesta tiene un código de estado |
| The user has admin privileges | El usuario tiene privilegios de admin |

### Frases cotidianas para programadores

| Inglés | Español |
|--------|---------|
| I have a question about the PR | Tengo una pregunta sobre el PR |
| She has experience with React | Ella tiene experiencia con React |
| We have a meeting at 3 PM | Tenemos una reunión a las 3 PM |
| Do you have access to the server? | ¿Tienes acceso al servidor? |
| I don't have the API key | No tengo la API key |

## To Do (Hacer)

### Conjugación en presente

| Sujeto | Verbo | Ejemplo |
|--------|-------|---------|
| I | do | I do code reviews on Mondays |
| You | do | You do great work |
| He/She/It | **does** | The script **does** the cleanup |
| We | do | We do daily standups |
| They | do | They do pair programming |

### Como verbo principal en programación

| Inglés | Español |
|--------|---------|
| The script does the backup | El script hace el respaldo |
| The middleware does the authentication | El middleware hace la autenticación |
| I do code reviews every morning | Yo hago code reviews cada mañana |
| What does this function do? | ¿Qué hace esta función? |
| The cron job does the cleanup | El cron job hace la limpieza |

---

## "Do/Does" como verbo auxiliar (para preguntas y negaciones)

Esta es la función más importante de "do/does" en inglés. Se usa para formar preguntas y negaciones con **todos los verbos excepto "to be"**.

### Formando preguntas

```
Do/Does + Sujeto + Verbo (forma base) + Complemento?
```

| Pregunta | Traducción |
|----------|------------|
| **Does** the function return a value? | ¿La función retorna un valor? |
| **Do** you have access to the repo? | ¿Tienes acceso al repositorio? |
| **Does** the API support pagination? | ¿La API soporta paginación? |
| **Do** the tests cover edge cases? | ¿Los tests cubren casos límite? |
| **Does** the server handle CORS? | ¿El servidor maneja CORS? |
| **Do** you use version control? | ¿Usas control de versiones? |
| **Does** this library support TypeScript? | ¿Esta librería soporta TypeScript? |

> **Regla clave:** Cuando usas "does" (tercera persona), el verbo principal pierde la -s. "Does the function return..." NO "Does the function returns..."

### Formando negaciones

```
Sujeto + do/does + not + Verbo (forma base) + Complemento
```

| Contracción | Ejemplo |
|-------------|---------|
| don't (do not) | I **don't** understand the error message |
| doesn't (does not) | The loop **doesn't** terminate |

| Negación | Traducción |
|----------|------------|
| The function doesn't return a value | La función no retorna un valor |
| I don't have the credentials | No tengo las credenciales |
| The loop doesn't terminate | El bucle no termina |
| We don't use jQuery anymore | Ya no usamos jQuery |
| The API doesn't accept XML | La API no acepta XML |
| She doesn't work on weekends | Ella no trabaja los fines de semana |
| The tests don't pass | Los tests no pasan |
| It doesn't compile | No compila |

## "Have" como auxiliar: "Have to" (Tener que)

"Have to" expresa obligación, similar a "tener que" en español.

| Inglés | Español |
|--------|---------|
| I have to fix this bug before the release | Tengo que arreglar este bug antes del release |
| You have to write unit tests | Tienes que escribir tests unitarios |
| She has to review the PR | Ella tiene que revisar el PR |
| We have to refactor this module | Tenemos que refactorizar este módulo |
| They have to update the dependencies | Tienen que actualizar las dependencias |

### Negativo e interrogativo de "have to"

| Forma | Ejemplo |
|-------|---------|
| Negativo | You **don't have to** deploy today (No tienes que desplegar hoy) |
| Pregunta | **Do** you **have to** use Docker? (¿Tienes que usar Docker?) |
| Negativo 3ra persona | She **doesn't have to** attend the meeting |
| Pregunta 3ra persona | **Does** he **have to** write documentation? |

## Errores comunes de hispanohablantes

| Error | Correcto | Explicación |
|-------|----------|-------------|
| ~~The function doesn't returns~~ | The function doesn't return | Con "doesn't", el verbo va en forma base |
| ~~Does the function returns?~~ | Does the function return? | Con "does", el verbo va en forma base |
| ~~I no have access~~ | I don't have access | Se necesita "do not" para negar |
| ~~Have you access?~~ | Do you have access? | Se necesita "Do" para preguntar |
| ~~The array have 5 items~~ | The array has 5 items | Tercera persona singular usa "has" |

## Ejercicios

### Ejercicio 1: Completa con have / has

1. The object ___ a method called "toString".
2. I ___ a question about the architecture.
3. The API ___ three endpoints.
4. We ___ access to the production database.
5. She ___ five years of experience.

### Ejercicio 2: Forma preguntas con Do / Does

1. The server supports WebSockets. → ___?
2. You understand the requirements. → ___?
3. The function accept parameters. → ___?
4. They use agile methodology. → ___?

### Ejercicio 3: Convierte a negativo con don't / doesn't

1. The code compiles correctly. → ___
2. I have the admin password. → ___
3. The function returns an error. → ___
4. We use that library. → ___

### Ejercicio 4: Traduce al inglés

1. ¿La función retorna un valor?
2. El arreglo tiene diez elementos.
3. El loop no termina.
4. ¿Tienes acceso al repositorio?
5. Tengo que arreglar este bug hoy.

---

### Respuestas

**Ejercicio 1:** 1. has, 2. have, 3. has, 4. have, 5. has

**Ejercicio 2:**
1. Does the server support WebSockets?
2. Do you understand the requirements?
3. Does the function accept parameters?
4. Do they use agile methodology?

**Ejercicio 3:**
1. The code doesn't compile correctly.
2. I don't have the admin password.
3. The function doesn't return an error.
4. We don't use that library.

**Ejercicio 4:**
1. Does the function return a value?
2. The array has ten elements.
3. The loop doesn't terminate.
4. Do you have access to the repository?
5. I have to fix this bug today.
