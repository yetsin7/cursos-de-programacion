# Chapter 15: Git and Versión Control

## Git y Control de versiones

Git es la herramienta de control de versiones mas utilizada en el mundo. Todo su
vocabulario esta en ingles, por lo que dominar estos términos es esencial para
cualquier desarrollador.

## Contenido

| Leccion | Tema | Descripción |
|---------|------|-------------|
| 01 | Git Commands Vocabulary | Comandos de Git con pronunciacion y ejemplos |
| 02 | Branching and Merging | Como hablar sobre ramas, merge y conflictos |
| 03 | Pull Request Language | Vocabulario para PRs y code reviews |
| 04 | Exercises | Ejercicios prácticos |

## Objetivos

Al completar este capítulo seras capaz de:

- Pronunciar y usar correctamente todos los comandos de Git
- Hablar sobre ramas y estrategias de merge en ingles
- Escribir y revisar Pull Requests con vocabulario profesional
- Participar en code reviews usando frases estandar de la industria
