# Capítulo 16: Fundamentos de Escritura Técnica

## Descripción

En este capítulo aprenderás los principios fundamentales de la escritura técnica en inglés.
La documentación clara y profesional es una habilidad esencial para cualquier programador.

## Contenido

| Lección | Tema | Descripción |
|---------|------|-------------|
| 01 | Active vs Passive Voice | Cuándo usar voz activa y pasiva en documentación |
| 02 | Clear and Concise Writing | Cómo escribir de forma clara, directa y sin relleno |
| 03 | Common Technical Phrases | Frases estándar que aparecen constantemente en docs |
| 04 | Exercises | Ejercicios prácticos del capítulo |

## Objetivos de aprendizaje

Al completar este capítulo serás capaz de:

- Escribir documentación técnica usando voz activa correctamente
- Eliminar palabras innecesarias y ser específico
- Usar frases técnicas estándar con naturalidad
- Redactar README, guías y comentarios de código profesionales

## Prerequisitos

- Capítulos 1-15 completados
- Familiaridad básica con documentación de software
