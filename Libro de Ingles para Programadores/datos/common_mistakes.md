# Common English Mistakes by Spanish-Speaking Programmers

## Los 50 errores mas comunes de ingles que cometen los programadores hispanohablantes

---

## Errores de vocabulario

| # | Error comun | Correcto | Explicacion |
|---|-------------|----------|-------------|
| 1 | "I want to **do** a function" | "I want to **create/write** a function" | "Do" es muy generico. Usa "create", "write" o "build" |
| 2 | "The code **works good**" | "The code **works well**" | "Good" es adjetivo, "well" es adverbio |
| 3 | "I will **do** a deploy" | "I will **do** a deployment" / "I will **deploy**" | "Deploy" como sustantivo es "deployment" |
| 4 | "**Actually** I'm working on..." | "**Currently** I'm working on..." | "Actually" = "en realidad", "currently" = "actualmente" |
| 5 | "I'm very **excited** about this bug" | "I'm very **frustrated** with this bug" | "Excited" = emocionado (positivo), no "excitado" |
| 6 | "The **library** is in the shelf" | "The **library** (software) / **book** is on the shelf" | En tech, "library" es una libreria de codigo |
| 7 | "I need to **support** the server" | "I need to **maintain/keep** the server (running)" | "Support" = soportar/apoyar, no "mantener" |
| 8 | "This is a **sensible** decision" | "This is a **sensitive** operation" | "Sensible" = razonable, "sensitive" = sensible |
| 9 | "The code is very **large**" | "The code is very **long**" / "The file is very **large**" | Codigo es "long" (largo), archivos son "large" (grandes) |
| 10 | "I will **assist** to the meeting" | "I will **attend** the meeting" | "Assist" = ayudar, "attend" = asistir (a un evento) |

## Errores de gramatica

| # | Error comun | Correcto | Explicacion |
|---|-------------|----------|-------------|
| 11 | "**It depends of** the config" | "**It depends on** the config" | Siempre "depends on", nunca "depends of" |
| 12 | "I'm **agree** with you" | "I **agree** with you" | "Agree" es verbo, no necesita "am" |
| 13 | "**The** JavaScript is popular" | "JavaScript is popular" | Los lenguajes de programacion no llevan articulo |
| 14 | "I have **20 years old**" | "I **am 20 years old**" | Edad usa "to be", no "to have" |
| 15 | "I want **that** you review my PR" | "I want **you to** review my PR" | Estructura "want + person + to + verb" |
| 16 | "**Is** a good framework" | "**It's** a good framework" | Falta el sujeto "it" |
| 17 | "I **am agree** with this approach" | "I **agree** with this approach" | "Agree" no necesita verbo auxiliar |
| 18 | "**The people is** complaining" | "**People are** complaining" | "People" es plural y no lleva "the" en general |
| 19 | "I **have** many experience" | "I **have** a lot of **experience**" | "Experience" es incontable |
| 20 | "**In** the last meeting" | "**At/During** the last meeting" | "At" para eventos, "in" para periodos |

## Errores de falsos amigos

| # | Error comun | Correcto | Explicacion |
|---|-------------|----------|-------------|
| 31 | "I will **realize** the deployment" | "I will **carry out/perform** the deployment" | "Realize" = darse cuenta, no "realizar" |
| 32 | "The **argument** with the client" | "The **discussion** with the client" | "Argument" suena a pelea. Usa "discussion" |
| 33 | "**Eventual** bugs will appear" | "**Possible/Potential** bugs may appear" | "Eventually" = finalmente, no "eventualmente" |
| 34 | "Let me **pretend** to fix this" | "Let me **try/attempt** to fix this" | "Pretend" = fingir, no "pretender" |
| 35 | "The **resume** of the meeting" | "The **summary** of the meeting" | "Resume" = curriculum, "summary" = resumen |
| 36 | "I need to **record** that..." | "I need to **remember** that..." | "Record" = grabar, "remember" = recordar |
| 37 | "This is very **actual**" | "This is very **current/up-to-date**" | "Actual" = real, "current" = actual |
| 38 | "The code was **molested** by changes" | "The code was **affected/modified** by changes" | "Molest" tiene significado muy diferente en ingles |
| 39 | "I will **assist** you with the code" | Correcto en este caso | "Assist" = ayudar (pero no = asistir a un lugar) |
| 40 | "The **fabric** of software" | "The **factory** pattern" | "Fabric" = tela, "factory" = fabrica |

## Errores de estructura de oraciones

| # | Error comun | Correcto | Explicacion |
|---|-------------|----------|-------------|
| 41 | "Is necessary to update" | "**It is** necessary to update" | El ingles necesita sujeto (it) |
| 42 | "Explain me how it works" | "Explain **to me** how it works" | "Explain to" + persona |
| 43 | "I suggest you **to use** React" | "I suggest you **use** React" | Despues de "suggest", subjuntivo (sin "to") |
| 44 | "**How** is called this method?" | "**What** is this method called?" | "What" para nombres, "how" para maneras |
| 45 | "I have the doubt if..." | "I'm **wondering** if/whether..." | "Doubt" es diferente. Usa "wondering" o "question" |

## Consejos generales

### Las 5 trampas mas peligrosas

1. **Falsos amigos**: "actually", "eventually", "realize", "pretend" no significan lo que parecen
2. **Preposiciones**: Son completamente diferentes al espanol. Memoriza las combinaciones
3. **Sujeto obligatorio**: En ingles SIEMPRE necesitas sujeto ("It is important", no "Is important")
4. **Contables vs incontables**: "information", "experience", "software", "code" son incontables
5. **Orden de palabras**: Adjetivo ANTES del sustantivo ("clean code", no "code clean")

### Palabras incontables comunes en tech

Estas palabras **nunca** llevan "a/an" ni plural con "s":

| Incorrecto | Correcto |
|-----------|----------|
| "a software" | "software" / "a piece of software" |
| "informations" | "information" |
| "an advice" | "advice" / "a piece of advice" |
| "feedbacks" | "feedback" |
| "a code" | "code" / "a piece of code" |
| "an equipment" | "equipment" / "a piece of equipment" |
| "a research" | "research" |
| "knowledges" | "knowledge" |
| "a homework" | "homework" |
| "a progress" | "progress" |
