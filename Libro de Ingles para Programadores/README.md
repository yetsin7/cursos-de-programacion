# 📘 Libro de Inglés para Programadores — De cero a NATIVO

> Repositorio gratuito, progresivo y en español para aprender **inglés orientado a la programación** desde nivel cero hasta nivel nativo profesional.

---

## ¿Que es esto?

El **Libro de Ingles para Programadores** es una coleccion de lecciones
organizadas como un libro tecnico interactivo. Su meta es que puedas leer,
escuchar, pronunciar y escribir ingles tecnico con confianza real.

- Explicaciones detalladas en **español**
- Vocabulario técnico real del mundo de la programación
- Pronunciación con guías fonéticas
- Ejemplos con código real
- Ejercicios prácticos con soluciones

Está diseñado para el mismo repositorio del **Libro de Flutter**, **Libro de JavaScript** y **Libro de Python**.

---

## ¿Para quien es?

Para **cualquier programador hispanohablante** que quiera:

- Aprender inglés desde cero con contexto de programación
- Leer documentación técnica en inglés con confianza
- Escribir código, comentarios y documentación en inglés profesional
- Comunicarse en equipos internacionales de desarrollo
- Prepararse para entrevistas técnicas en inglés
- Alcanzar un nivel nativo en comunicación técnica

No se requiere experiencia previa con ingles. Solo ganas de aprender y practica
constante.

## Que vas a lograr

Este libro no se limita a memorizar vocabulario. Busca que puedas:

- leer documentacion tecnica sin traducir cada palabra;
- entender mensajes de error;
- escribir comentarios y documentacion en ingles claro;
- comunicar ideas tecnicas en equipos internacionales;
- pronunciar mejor nombres de herramientas, conceptos y comandos.

## ¿Cómo usar este libro?

1. **Ve en orden** — cada capítulo construye sobre el anterior
2. **Lee las explicaciones** — están en español para que entiendas todo
3. **Practica la pronunciación** — usa las guías fonéticas incluidas
4. **Haz los ejercicios** — están al final de cada lección
5. **Aplícalo al código** — cada lección tiene ejemplos con código real
