# Common Email Templates - Plantillas comunes de email

## Introduccion

Tener plantillas de email listas te ahorra tiempo y te asegura mantener un tono profesional consistente. Aqui encontraras plantillas para las situaciones mas comunes que enfrentaras como desarrollador en un equipo internacional.

---

## 1. Solicitar acceso (Requesting access)

```
Subject: Request: Access to [system/environment]

Hi [Name],

I'd like to request access to [system name]. I need it 
to [specific reason].

Here are the details:
- Username/email: [your-email@company.com]
- Access level needed: [read-only / read-write / admin]
- Project: [project name]
- Approved by: [manager name, if applicable]

Please let me know if you need any additional information 
or if there's a ticket I should file.

Thanks,
[Your name]
```

---

## 2. Reportar un bug (Reporting a bug)

```
Subject: Bug report: [Brief description of the issue]

Hi [Name/Team],

I found an issue in [system/feature] that I'd like to report.

**Environment:** [production / staging / local]
**Browser/OS:** [Chrome 120 / macOS 14.2]
**Steps to reproduce:**
1. Navigate to [page/endpoint]
2. Click on [element] / Send [request]
3. Observe [unexpected behavior]

**Expected behavior:** [What should happen]
**Actual behavior:** [What actually happens]

**Screenshot/logs:** [Attached or linked]

I've also created a ticket: [JIRA-123 / GitHub Issue #456]

Let me know if you need more details.

Thanks,
[Your name]
```

## 3. Proponer un cambio tecnico (Proposing a technical change)

```
Subject: Proposal: [Brief description of the change]

Hi team,

I'd like to propose a change to [system/component].

**Current situation:**
[Describe the current state and its limitations]

**Proposed change:**
[Describe what you want to change and how]

**Benefits:**
- [Benefit 1]
- [Benefit 2]
- [Benefit 3]

**Risks and mitigation:**
- [Risk 1] — mitigated by [approach]

**Estimated effort:** [X days/sprints]

I've drafted a more detailed document here: [link]

I'd love to get your thoughts on this. Would it make sense 
to discuss it in our next team meeting?

Best,
[Your name]
```

## 4. Pedir extension de deadline (Asking for a deadline extension)

```
Subject: Request: Extension for [task/project name]

Hi [Manager's name],

I wanted to give you an early heads-up that I may need 
additional time for [task/project name], originally due 
on [date].

**Reason:** [Brief, honest explanation — e.g., "The scope 
turned out to be larger than estimated" or "I discovered 
an unexpected dependency on the auth service."]

**Current progress:** [What's done so far — e.g., "The core 
functionality is complete. I still need to finish testing 
and documentation."]

**New estimated completion:** [Proposed new date]

**What I'm doing to minimize delay:**
- [Action 1]
- [Action 2]

I apologize for the delay and want to make sure we deliver 
quality work. I'm happy to discuss priorities if we need 
to adjust the scope instead.

Thanks for your understanding,
[Your name]
```

## 5. Actualizacion de estado (Status update)

```
Subject: Status update: [Project name] — Week of [date]

Hi team,

Here's the weekly update for [project name]:

**Completed this week:**
- Implemented user authentication flow
- Fixed bug with session timeout (JIRA-789)
- Code review for PR #234

**In progress:**
- Database migration for new user fields (70% done)
- Performance testing for the search API

**Blocked:**
- Waiting for design approval on the settings page

**Next week's plan:**
- Complete database migration
- Start integration testing
- Deploy to staging

Let me know if you have any questions.

Best,
[Your name]
```

## 6. Presentarse a un nuevo equipo (Introducing yourself)

```
Subject: Introduction: [Your name] — New [role]

Hi everyone,

I'm [Your name], and I'm excited to be joining the team 
as a [role] starting [date].

A little about me: I have [X] years of experience in 
[technologies/areas]. Most recently, I was at [Company] 
where I worked on [brief description]. I'm passionate 
about [area of interest].

Outside of work, I enjoy [1-2 personal interests — keeps 
it human and approachable].

I'm looking forward to getting to know everyone and 
contributing to [project/product]. Feel free to reach 
out — I'd love to connect!

Best,
[Your name]
```

## 7. Mensaje de fuera de oficina (Out-of-office)

```
Subject: Out of Office: [Your name]

// Version automatica (auto-reply):

Hi,

Thank you for your email. I'm currently out of the office 
from [start date] to [end date] with limited access to email.

For urgent matters, please contact [colleague name] at 
[email]. I'll respond to your email when I return.

Thanks,
[Your name]

// Version mas informal:

Hey there! I'm out of the office until [date], recharging 
my batteries. I'll get back to you when I return. If it's 
urgent, reach out to [name] at [email].

Cheers,
[Your name]
```

## 8. Carta de renuncia (Resignation letter)

```
Subject: Resignation — [Your name]

Dear [Manager's name],

I am writing to formally notify you of my resignation 
from my position as [job title] at [company name], 
effective [last working day — typically 2 weeks from now].

I have thoroughly enjoyed my time at [company] and am 
grateful for the opportunities I've had to grow 
professionally. Working on [specific project or with 
specific team] has been a highlight of my career.

I am committed to ensuring a smooth transition. I'm 
happy to help document my current projects, train my 
replacement, and complete any outstanding work during 
my remaining time.

Thank you for your support and guidance over the past 
[duration]. I wish you and the team continued success.

Sincerely,
[Your full name]
```

## Frases utiles para cualquier email

### Para ser diplomatico

| Situacion | Frase |
|-----------|-------|
| Recordar algo pendiente | *"Just a friendly reminder that..."* |
| Decir no educadamente | *"Unfortunately, I won't be able to..."* |
| Pedir algo con tacto | *"Would it be possible to..."* |
| Senalar un error | *"I noticed a small discrepancy in..."* |
| Expresar urgencia sin sonar agresivo | *"I'd appreciate it if we could address this soon."* |

### Para seguimiento (follow-up)

| Situacion | Frase |
|-----------|-------|
| Primer seguimiento | *"Following up on my previous email..."* |
| Segundo seguimiento | *"Just circling back on this..."* |
| Con urgencia | *"I wanted to check in on the status of..."* |
| Sin urgencia | *"No rush, but when you get a chance..."* |

---

## Consejo final

> Guarda estas plantillas y personaliza los detalles para cada situacion. Con el tiempo, tendras tu propia coleccion de templates que puedes adaptar en minutos. Un email bien escrito no solo comunica informacion — demuestra profesionalismo.
