# Chapter 14: Error Messages and Debugging

## Mensajes de Error y Depuracion

Los errores son parte fundamental del desarrollo de software. En este capitulo
aprenderemos el vocabulario necesario para entender tipos de errores, describir
bugs profesionalmente y comunicar problemas en ingles.

## Contenido

| Leccion | Tema | Descripcion |
|---------|------|-------------|
| 01 | Common Error Types | Tipos de errores comunes y su significado |
| 02 | Debugging Vocabulary | Vocabulario de depuracion y herramientas |
| 03 | Describing Bugs | Como describir bugs en ingles profesional |
| 04 | Exercises | Ejercicios practicos |

## Objetivos

Al completar este capitulo seras capaz de:

- Identificar y explicar los tipos de errores mas comunes
- Usar vocabulario de debugging con confianza
- Escribir reportes de bugs claros y profesionales en ingles
- Comunicar problemas tecnicos a tu equipo en ingles
