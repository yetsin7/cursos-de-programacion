# Chapter 25: Advanced Technical Writing

## Descripción

Este capítulo cubre la escritura técnica avanzada en ingles: RFCs, ADRs, especificaciones técnicas e informes de incidentes. Aprenderas a redactar documentos profesionales que comunican decisiones técnicas con claridad y precision.

## Contenido

| Leccion | Tema | Archivo |
|---------|------|---------|
| 1 | Escribir RFCs y ADRs | `01_writing_rfcs_and_adrs.md` |
| 2 | Especificaciones técnicas | `02_writing_technical_specs.md` |
| 3 | Informes de incidentes | `03_writing_incident_reports.md` |
| 4 | Ejercicios | `04_exercises.md` |

## Objetivos de aprendizaje

- Redactar RFCs y ADRs con estructura y lenguaje profesional
- Escribir especificaciones técnicas claras y completas
- Crear informes de incidentes (postmortems) efectivos
- Usar el vocabulario y las frases estandar de la industria
