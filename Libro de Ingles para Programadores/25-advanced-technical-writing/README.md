# Chapter 25: Advanced Technical Writing

## Descripcion

Este capitulo cubre la escritura tecnica avanzada en ingles: RFCs, ADRs, especificaciones tecnicas e informes de incidentes. Aprenderas a redactar documentos profesionales que comunican decisiones tecnicas con claridad y precision.

## Contenido

| Leccion | Tema | Archivo |
|---------|------|---------|
| 1 | Escribir RFCs y ADRs | `01_writing_rfcs_and_adrs.md` |
| 2 | Especificaciones tecnicas | `02_writing_technical_specs.md` |
| 3 | Informes de incidentes | `03_writing_incident_reports.md` |
| 4 | Ejercicios | `04_exercises.md` |

## Objetivos de aprendizaje

- Redactar RFCs y ADRs con estructura y lenguaje profesional
- Escribir especificaciones tecnicas claras y completas
- Crear informes de incidentes (postmortems) efectivos
- Usar el vocabulario y las frases estandar de la industria
