# Capítulo 20: Gramática Avanzada

## Descripción

Este capítulo cubre estructuras gramaticales avanzadas del inglés que
aparecen constantemente en el entorno de programación: condicionales,
verbos modales, cláusulas relativas y reported speech.

## Contenido

| Lección | Tema | Descripción |
|---------|------|-------------|
| 01 | Conditionals | Los 4 tipos de condicionales en contexto tech |
| 02 | Modal Verbs | can, could, should, must, might, would en tech |
| 03 | Relative Clauses | who, which, that, where, whose en tech |
| 04 | Reported Speech | Cómo reportar lo que otros dijeron (reuniones, standups) |
| 05 | Exercises | Ejercicios prácticos del capítulo |

## Objetivos de aprendizaje

Al completar este capítulo serás capaz de:

- Usar los 4 tipos de condicionales correctamente
- Aplicar verbos modales con el grado correcto de certeza y cortesía
- Construir cláusulas relativas naturales
- Reportar conversaciones de reuniones y standups

## Prerequisitos

- Capítulos 1-19 completados
