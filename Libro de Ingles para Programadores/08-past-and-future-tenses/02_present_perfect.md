# Present Perfect

## Presente perfecto en contexto de programación

El presente perfecto conecta el pasado con el presente. Se usa para hablar de acciones completadas que tienen **relevancia ahora**, experiencias acumuladas, o acciones recientes sin especificar cuándo exactamente ocurrieron.

---

## 1. Estructura

```
Sujeto + have/has + participio pasado
```

- **have** con: I, you, we, they
- **has** con: he, she, it

### Afirmativa

```text
I have deployed the fix.          (He desplegado el fix)
She has reviewed the PR.          (Ella ha revisado el PR)
We have updated the dependencies. (Hemos actualizado las dependencias)
The build has failed.             (El build ha fallado)
```

### Contracciones (muy comunes al hablar)

```text
I've deployed the fix.
She's reviewed the PR.
We've updated the dependencies.
It's failed three times.
```

### Negativa

```text
I haven't finished the refactoring yet.
She hasn't pushed her changes.
We haven't tested this on production.
The pipeline hasn't run today.
```

### Pregunta

```text
Have you tested this locally?
Has she merged the branch?
Have they responded to the issue?
Has the build completed?
```

---

## 2. Cuándo usar presente perfecto

### 2.1 Acciones recientes con relevancia al presente

La acción acaba de ocurrir y el resultado importa ahora.

```text
I've deployed the fix.              (El fix ya está en producción)
She's just pushed her changes.      (Los cambios ya están en el repo)
The build has failed.               (No podemos hacer release ahora)
I've fixed the bug.                 (El bug ya no existe)
```

### 2.2 Experiencias (alguna vez / nunca)

Con **ever** (alguna vez) y **never** (nunca):

```text
Have you ever used Kubernetes?       (¿Alguna vez has usado Kubernetes?)
I've never worked with GraphQL.      (Nunca he trabajado con GraphQL)
She's never seen this error before.  (Ella nunca ha visto este error antes)
Have you ever deployed to AWS?       (¿Alguna vez has desplegado en AWS?)
We've never had a production outage. (Nunca hemos tenido una caída en producción)
```

### 2.3 Acciones repetidas hasta ahora

```text
The build has failed three times today.
I've reviewed five PRs this week.
We've deployed twelve hotfixes this month.
She's written over 200 unit tests.
```

### 2.4 Acciones que empezaron en el pasado y continúan

Con **for** (duración) y **since** (punto de inicio):

```text
I've worked here for two years.
We've used React since 2022.
She's been on the team since January.
The server has been running for 90 days.
```

## 3. Palabras clave del presente perfecto

| Palabra | Significado | Posición | Ejemplo |
|---------|-------------|----------|---------|
| just | acabar de | antes del participio | I've just finished the migration. |
| already | ya | antes del participio | We've already fixed that. |
| yet | todavía / ya (preguntas) | al final | I haven't tested it yet. / Have you finished yet? |
| ever | alguna vez | antes del participio | Have you ever used Docker? |
| never | nunca | antes del participio | I've never deployed to Azure. |
| recently | recientemente | al final o inicio | We've updated the API recently. |
| so far | hasta ahora | al final | No issues so far. |
| since | desde | + punto en el tiempo | I've used Linux since 2020. |
| for | durante | + duración | She's worked here for three years. |

## 4. Presente perfecto vs. pasado simple

Esta es la diferencia más importante y confusa para hispanohablantes.

| Presente perfecto | Pasado simple |
|-------------------|---------------|
| No se especifica cuándo | Se especifica cuándo |
| Relevante al presente | Acción terminada en el pasado |
| Experiencia general | Evento específico |

### Comparación directa

| Presente perfecto | Pasado simple |
|-------------------|---------------|
| I've fixed the bug. (resultado: está arreglado) | I fixed the bug yesterday. (cuándo: ayer) |
| She's deployed three times. (hasta ahora) | She deployed at 3 PM. (momento específico) |
| Have you ever used Go? (experiencia general) | Did you use Go in that project? (proyecto específico) |
| We've had issues with the API. (sigue siendo relevante) | We had issues last week. (la semana pasada, terminado) |
| I haven't finished yet. (todavía en progreso) | I didn't finish on time. (plazo ya pasó) |

### Regla práctica

```text
Si mencionas CUÁNDO (yesterday, last week, at 3 PM, in 2023):
  -> Usa PASADO SIMPLE: "I fixed the bug yesterday."

Si NO mencionas cuándo, y el resultado importa AHORA:
  -> Usa PRESENTE PERFECTO: "I've fixed the bug." (ya está arreglado)
```

## 6. Ejemplos en el trabajo diario

### En standups

```text
I've finished the login feature.
I've created a PR for the search module.
I haven't started the database migration yet.
I've been working on the API refactoring.
```

### En code reviews

```text
I've left some comments on the PR.
Have you addressed the feedback?
She's approved the changes.
We haven't received the review yet.
```

### En reportes de estado

```text
We've completed 8 out of 10 stories this sprint.
The team has resolved all critical bugs.
We haven't met the performance targets yet.
The migration has been running smoothly so far.
```

## 7. Errores comunes de hispanohablantes

| Incorrecto | Correcto | Explicación |
|------------|----------|-------------|
| I have fix the bug. | I have fixed the bug. | Falta el participio pasado |
| I've deployed yesterday. | I deployed yesterday. | Con "yesterday" se usa pasado simple |
| She have reviewed the PR. | She has reviewed the PR. | Tercera persona usa "has" |
| Did you ever used Go? | Have you ever used Go? | "Ever" va con presente perfecto |
| I have went to the meeting. | I have gone to the meeting. | "Gone" es el participio, no "went" |
