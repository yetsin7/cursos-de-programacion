# Future Tenses

## Tiempos futuros en contexto de programación

En inglés hay varias formas de hablar del futuro. Cada una tiene un matiz diferente que es importante conocer para planificar sprints, hacer promesas, expresar intenciones y describir planes en tu equipo de desarrollo.

---

## 1. Will — Decisiones, predicciones y promesas

### Estructura

```
Sujeto + will + verbo base
```

### Cuándo usar will

| Uso | Ejemplo | Explicación |
|-----|---------|-------------|
| Decisión instantánea | I'll fix it right now. | Decido en el momento |
| Promesa | I will have the PR ready by tomorrow. | Me comprometo |
| Predicción | This feature will be deprecated in v5. | Creo que pasará |
| Ofrecimiento | I'll take a look at the bug. | Me ofrezco |
| Hechos futuros | The release will happen on March 15. | Dato confirmado |

### Ejemplos en programación

```text
I will fix it tomorrow.
The migration will take about two hours.
This approach will cause performance issues.
We will need to refactor this module eventually.
Don't worry, I'll handle the deployment.
The new API will support pagination.
```

### Negativa: won't (will not)

```text
This won't work in production.
I won't be able to finish by Friday.
The tests won't pass without the mock data.
The feature won't be included in this release.
```

### Pregunta

```text
Will the build finish in time?
Will you review my PR today?
Will this breaking change affect other teams?
When will the new version be available?
```

## 2. Going to — Planes e intenciones

### Estructura

```
Sujeto + am/is/are + going to + verbo base
```

### Cuándo usar going to

| Uso | Ejemplo | Explicación |
|-----|---------|-------------|
| Plan decidido | We're going to refactor this module. | Ya lo decidimos |
| Intención | I'm going to learn Rust this year. | Tengo la intención |
| Predicción con evidencia | The server is going to crash — look at the memory usage. | Veo señales |

### Ejemplos en programación

```text
We're going to migrate to TypeScript next quarter.
I'm going to write tests for this module.
She's going to present the architecture at the meeting.
They're going to deprecate the old API.
The team is going to adopt a new CI/CD tool.
I'm going to refactor the authentication flow.
```

### Negativa

```text
We're not going to deploy on Friday.
I'm not going to use that library — it's unmaintained.
They're not going to approve the budget for new servers.
```

### Pregunta

```text
Are you going to push today?
Is the team going to use Docker?
When are we going to start the migration?
```

## 3. Presente continuo para el futuro — Planes confirmados

Usamos el presente continuo (**am/is/are + -ing**) para eventos futuros que ya están organizados o confirmados (como reuniones, fechas de release).

### Ejemplos

```text
The meeting is starting at 3 PM.
We're deploying tomorrow morning.
She's presenting at the conference next week.
I'm pairing with Carlos on Thursday.
The sprint is ending on Friday.
They're releasing version 3.0 next month.
We're having a retrospective this afternoon.
```

> **Diferencia con "going to":**
> - "Going to" = intención o plan decidido
> - Presente continuo = ya está agendado, organizado, confirmado

```text
I'm going to learn Docker. (intención, no sé cuándo)
I'm taking a Docker course next Monday. (ya está agendado)
```

## 4. Comparación de los tres futuros

| Forma | Uso principal | Ejemplo |
|-------|--------------|---------|
| **will** | Decisión en el momento, promesa, predicción | I'll fix the bug right now. |
| **going to** | Plan decidido, intención, predicción con evidencia | We're going to refactor this. |
| **Presente continuo** | Evento confirmado/agendado | We're deploying at 5 PM. |

### Ejemplo de las tres formas en contexto

```text
Escenario: un bug en producción

1. [will - decisión inmediata]
   "There's a bug in production. I'll take a look."

2. [going to - plan formado]
   "I found the issue. I'm going to write a fix this afternoon."

3. [presente continuo - ya agendado]
   "I'm deploying the hotfix at 4 PM."
```

## 5. Otras formas de expresar futuro

### Will be + -ing (futuro continuo)

Describe una acción en progreso en un momento futuro.

```text
I'll be working on the migration all day tomorrow.
This time next week, we'll be running the new system.
She'll be presenting at 3 PM, so she won't be available.
```

### Will have + participio (futuro perfecto)

Describe una acción que estará completada antes de un momento futuro.

```text
By Friday, I will have finished the module.
By next sprint, we will have migrated all services.
By the end of the year, the team will have shipped 50 features.
```

## 6. Expresiones de tiempo para el futuro

| Expresión | Significado | Ejemplo |
|-----------|-------------|---------|
| tomorrow | mañana | I'll push it tomorrow. |
| next week/month/year | la próxima semana/mes/año | We're releasing next week. |
| in two days | en dos días | It'll be ready in two days. |
| by Friday | para el viernes | I'll have it done by Friday. |
| this afternoon | esta tarde | We're meeting this afternoon. |
| soon | pronto | The fix will be available soon. |
| later | más tarde | I'll review it later. |
| eventually | eventualmente | We'll eventually rewrite it. |
| at some point | en algún momento | We'll need to address this at some point. |

---

## 7. Frases comunes del futuro en el trabajo diario

### En standups

```text
Today I'm going to work on the search feature.
I'll finish the PR review this morning.
I'm pairing with Ana this afternoon.
I'll have the hotfix ready by noon.
```

### En planificación

```text
We're going to implement the new payment flow next sprint.
The release is happening on March 20th.
We'll need to allocate more time for testing.
By Q3, we will have migrated all microservices.
```

### En conversaciones informales

```text
I'll send you the link later.
We're going to need more developers for this.
I think this approach will work.
The deadline is going to be tight.
```

## 8. Errores comunes de hispanohablantes

| Incorrecto | Correcto | Explicación |
|------------|----------|-------------|
| I will to fix it. | I will fix it. | No se usa "to" después de "will" |
| I going to deploy. | I'm going to deploy. | Falta el verbo "am" |
| I will can finish. | I will be able to finish. | No se combinan dos modales |
| Tomorrow I deploy. | Tomorrow I'm deploying. / I'll deploy tomorrow. | Falta forma de futuro |
| We will going to | We're going to / We will | No se mezclan will y going to |

---

## Resumen

| Forma | Estructura | Uso | Ejemplo |
|-------|-----------|-----|---------|
| Will | will + base | Decisión, promesa, predicción | I'll fix it now. |
| Going to | am/is/are going to + base | Plan, intención | We're going to refactor this. |
| Presente continuo | am/is/are + -ing | Evento confirmado | We're deploying at 5 PM. |
| Futuro continuo | will be + -ing | En progreso en el futuro | I'll be working on it all day. |
| Futuro perfecto | will have + participio | Completado antes de X | By Friday, I'll have finished. |
