# Chapter 13: Writing Comments and Documentation

## Escritura de Comentarios y Documentación

Escribir comentarios claros y documentación profesional en ingles es una habilidad
esencial para cualquier desarrollador que trabaje en equipos internacionales o
proyectos open source.

## Contenido

| Leccion | Tema | Descripción |
|---------|------|-------------|
| 01 | Writing Code Comments | Comentarios claros, convenciones TODO/FIXME, JSDoc, Dartdoc |
| 02 | Writing README Files | Estructura de un buen README profesional |
| 03 | Writing Commit Messages | Conventional commits y mensajes efectivos |
| 04 | Writing Documentation | Documentación técnica: tutoriales, guías, referencia |
| 05 | Exercises | Ejercicios prácticos de escritura |

## Objetivos

Al completar este capítulo seras capaz de:

- Escribir comentarios de código claros y útiles en ingles
- Crear archivos README profesionales
- Redactar mensajes de commit siguiendo convenciones estandar
- Escribir documentación técnica clara y organizada
