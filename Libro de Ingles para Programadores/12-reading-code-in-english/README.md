# Chapter 12: Reading Code in English

## Lectura de Código en Ingles

Saber leer código en voz alta en ingles es una habilidad crucial para participar en
pair programming, code reviews, entrevistas técnicas y reuniones de equipo.

## Contenido

| Leccion | Tema | Descripción |
|---------|------|-------------|
| 01 | Reading Variable Names | Como leer camelCase, snake_case, PascalCase y mas |
| 02 | Reading Code Aloud | Como leer operadores, simbolos y expresiones en voz alta |
| 03 | Reading Error Messages | Como leer y entender mensajes de error comúnes |
| 04 | Exercises | Ejercicios prácticos de lectura de código |

## Objetivos

Al completar este capítulo seras capaz de:

- Leer cualquier nombre de variable en voz alta correctamente
- Pronunciar operadores y simbolos de programación en ingles
- Entender y explicar mensajes de error comúnes
- Comunicar código verbalmente en reuniones y pair programming
