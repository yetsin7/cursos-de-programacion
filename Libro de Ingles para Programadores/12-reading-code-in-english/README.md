# Chapter 12: Reading Code in English

## Lectura de Codigo en Ingles

Saber leer codigo en voz alta en ingles es una habilidad crucial para participar en
pair programming, code reviews, entrevistas tecnicas y reuniones de equipo.

## Contenido

| Leccion | Tema | Descripcion |
|---------|------|-------------|
| 01 | Reading Variable Names | Como leer camelCase, snake_case, PascalCase y mas |
| 02 | Reading Code Aloud | Como leer operadores, simbolos y expresiones en voz alta |
| 03 | Reading Error Messages | Como leer y entender mensajes de error comunes |
| 04 | Exercises | Ejercicios practicos de lectura de codigo |

## Objetivos

Al completar este capitulo seras capaz de:

- Leer cualquier nombre de variable en voz alta correctamente
- Pronunciar operadores y simbolos de programacion en ingles
- Entender y explicar mensajes de error comunes
- Comunicar codigo verbalmente en reuniones y pair programming
