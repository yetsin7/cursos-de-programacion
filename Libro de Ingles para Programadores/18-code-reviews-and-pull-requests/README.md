# Capítulo 18: Code Reviews y Pull Requests

## Descripción

Este capítulo te enseña el inglés necesario para participar efectivamente
en code reviews y pull requests. Aprenderás a dar feedback constructivo,
responder a comentarios y escribir descripciones de PR profesionales.

## Contenido

| Lección | Tema | Descripción |
|---------|------|-------------|
| 01 | Giving Code Review Feedback | Cómo dar feedback constructivo y profesional |
| 02 | Receiving and Responding | Cómo recibir feedback y responder apropiadamente |
| 03 | PR Templates and Descriptions | Cómo escribir descripciones de PR efectivas |
| 04 | Exercises | Ejercicios prácticos del capítulo |

## Objetivos de aprendizaje

Al completar este capítulo serás capaz de:

- Dar feedback en code reviews con el tono correcto
- Responder a comentarios de forma profesional
- Escribir descripciones de PR claras y completas
- Usar las convenciones y frases estándar de la industria

## Prerequisitos

- Capítulos 1-17 completados
- Experiencia básica con Git y pull requests
