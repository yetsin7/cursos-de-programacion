# Chapter 29: Final Project - Professional Profile

## Descripcion

Este es el capitulo final del libro. Aqui aplicaras todo lo aprendido para crear tu perfil profesional completo en ingles: LinkedIn, GitHub, resume/CV y cover letter. Al terminar, tendras todos los materiales necesarios para aplicar a trabajos tech internacionales.

## Contenido

| Leccion | Tema | Descripcion |
|---------|------|-------------|
| 01 | LinkedIn Profile | Como escribir un perfil de LinkedIn profesional |
| 02 | GitHub Profile | Como crear un perfil de GitHub destacado |
| 03 | Resume / CV | Como escribir un CV tecnico en ingles |
| 04 | Cover Letter | Como escribir una carta de presentacion |
| 05 | Final Exercises | Proyecto final: tu perfil profesional completo |

## Objetivos de aprendizaje

- Escribir un perfil de LinkedIn atractivo para reclutadores internacionales
- Crear un perfil de GitHub que demuestre tus habilidades
- Redactar un resume/CV que pase filtros ATS
- Componer cover letters persuasivas para empresas tech
