# Capítulo 03 — Crear tu cuenta en GitHub

En este capítulo aprenderás a crear tu cuenta de GitHub, configurar tu perfil
profesionalmente y activar las opciones de seguridad más importantes.

GitHub es GRATIS para uso individual. No necesitas pagar nada para seguir este curso.

---

## ¿Qué es una cuenta de GitHub?

Tu cuenta de GitHub es tu identidad como desarrollador en internet. Es como tu
portafolio profesional: otros programadores, reclutadores y empresas pueden ver
tus proyectos, tu actividad y tus contribuciones.

Muchas empresas revisan tu perfil de GitHub como parte del proceso de contratación.
Por eso es importante tener un perfil bien configurado y activo.

---

## ¿Qué necesitas para crear una cuenta?

- Un correo electrónico (el mismo que configuraste en Git)
- Un nombre de usuario (elige uno profesional, será tu identidad pública)
- Una contraseña segura

---

## Elementos importantes de tu perfil

- **Username** — tu nombre de usuario público (aparece en la URL: github.com/tu-username)
- **Avatar** — tu foto de perfil (usa una foto profesional o un avatar reconocible)
- **Bio** — una descripción corta sobre ti (qué haces, qué te interesa)
- **README de perfil** — un archivo especial que aparece en tu página de perfil
- **Pinned repositories** — repos destacados que quieres que la gente vea primero

## Conceptos clave de este capítulo

- **GitHub account** — tu cuenta personal en la plataforma
- **Username** — nombre único que te identifica en GitHub
- **Profile README** — archivo Markdown especial que aparece en tu perfil
- **2FA (Two-Factor Authentication)** — capa extra de seguridad para tu cuenta
- **SSH key** — clave criptográfica para conectarte a GitHub sin contraseña
