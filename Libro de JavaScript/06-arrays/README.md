# Módulo 06 — Arrays en JavaScript

Arrays en JS: la estructura de datos más usada. Domina los métodos funcionales
y el principio de inmutabilidad para escribir código predecible y limpio.

```bash
node 01_arrays_basico.js
node 02_metodos_funcionales.js
node 03_ordenar_y_buscar.js
node 04_destructuring.js
node 05_ejercicios.js
```

## Conceptos clave

- Los arrays son **objetos** en JavaScript (typeof [] === 'object')
- Diferencia entre métodos **mutantes** (modifican el original) y **no mutantes**
- Los métodos funcionales (map, filter, reduce) son la base del JS moderno
- Preferir **inmutabilidad**: crear nuevos arrays en vez de mutar los existentes
