# 17 — npm y Herramientas del Ecosistema

npm (Node Package Manager) es el gestor de paquetes de Node.js y el repositorio más grande de código abierto del mundo.

## Flujo típico de un proyecto

```bash
# 1. Inicializar el proyecto
npm init -y

# 2. Instalar dependencias
npm install express
npm install -D nodemon eslint prettier

# 3. Correr el proyecto en desarrollo
npm run dev
```
